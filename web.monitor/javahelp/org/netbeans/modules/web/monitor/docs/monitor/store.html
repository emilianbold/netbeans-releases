Saving HTTP Monitor Data Records

See Also

You can save monitor data records in the Current Records subcategory of the All Records tree view. This action lets you view or replay the monitor data records in a later IDE session.

To save a current monitor data record:

  1. Right-click the monitor data record to be saved. You can select multiple monitor data records at one time by using the Shift or Control key.
  2. Choose Save from the contextual menu.
    3. The selected monitor data records are moved, not copied, to the Saved Records subcategory.
