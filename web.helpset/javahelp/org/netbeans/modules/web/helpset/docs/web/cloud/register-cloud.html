Registering a Cloud Account

See Also 

To develop and deploy applications to cloud-based servers you need to register your cloud account with the IDE. To register your cloud account open the Services window (Ctrl-5) and perform the following steps.

  1. Right-click the Cloud node in the Services window and select Add Cloud to open the Add Cloud Provider wizard.
  2. Select a cloud provider and click Next.
  3. Specify the details of your cloud account. Click Next.

    When you click Next the IDE attempts to communicate with your cloud account. If communication is successful a list of the resources associated with your cloud account is displayed in the wizard.

  4. Click Finish.

After registering a cloud account, the web/application servers associated with the account appear in the Services window under the Servers node.
