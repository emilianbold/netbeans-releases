Cloud Providers Manager

In the Cloud Providers manager, you can add a cloud provider, remove a cloud provider, or change some details of a cloud provider. You open the Cloud Providers manager by choosing Tools > Cloud Providers from the main menu.

Add Cloud/Remove Cloud

To add a cloud provider, select Add Cloud. You can add either an Amazon Elastic Beanstalk or an Oracle Cloud provider (plugin required). To remove a cloud provider, select a provider from the list and click Remove Cloud.

Change Amazon Elastic Beanstalk Properties

If you select an Amazon Elastic Beanstalk provider from the list, you can change the public key and secret key.

Change Oracle Cloud Properties

If you select an Oracle Cloud provider from the list, you can edit the following properties:

  • Database Service Name. The name of the Database Service you created when you set up your Oracle Cloud account.
  • SDK. Path to your local copy of the Oracle Cloud SDK. Click Configure to browse for this file.
  • Administrator. Your identity as Oracle Cloud administrator (the email you gave when you set up your Oracle Cloud account).
  • Password. Your Oracle Cloud administrator password.

The other properties in the dialog box are read-only.
