New Web Application Wizard: Spring Framework

See Also 

In the fourth panel of the New Web Application wizard, if you select Spring as your framework, you can modify the Spring configuration. The panel displays the following configuration options:

  • Dispatcher Name. Specify the name of the Spring dispatcher servlet.
  • Dispatcher Mapping. Specify the mapping of the dispatcher. The default is set to *.htm.
In the Libraries tab, you can specify whether JSTLs will be included in the project.
