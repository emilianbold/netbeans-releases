About the IDE's Technology Version Support

See Also

The IDE supports the following versions for web application development. However, the IDE compiles your applications against the Java EE specification supported by your target server. For GlassFish 3, this is the Java EE 6 specification (JSR 316). For GlassFish v2, this is the Java EE 5 specification (JSR 244). For Tomcat 6.x, these are the Servlet 2.5 (JSR 154) and JSP 2.1 (JSR 245) specifications.

    • JSR 316: Java Platform, Enterprise Edition 6 (Java EE 6):
