Defining Database Explorer Debug Mode

See Also

Enable debug mode to see SQL commands that are being sent to the database. By default, debug mode is disabled.

To enable debug mode:

  • Right-click the Databases node in the Runtime window, and choose Enable Debug from the contextual menu.

    Once debug mode is enabled, the IDE sends SQL commands to the Output window, where you can view the command log. Debug mode does not list all SQL commands, only those which you cannot see. For example, SQL queries for viewing data are not logged.

To disable debug mode:

  • Right-click the Databases node in the Runtime window, and choose Disable Debug from the contextual menu.
    • You can also enable or disable debug mode from the Options window. Choose Tools  Options. Then expand IDE Configuration  Server and External Tool Settings and select the Database Explorer node. Select the Debug Mode checkbox to enable the property, or deselect the checkbox to disable the property.
