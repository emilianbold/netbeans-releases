Ruby Project Properties: Sources Options

See Also

Use this dialog box to specify the source and test directories for your Ruby project. You open the dialog box by right-clicking the main project node for a Ruby project in the Projects window, choosing Properties, and then clicking the Sources node in the Project Properties dialog box.

  • You can use the buttons to the right of the directory lists to add, remove, and change the order of the directories.
  • Encoding. Sets the character encoding for your application, which is used by the IDE when writing files to disk. The default encoding is utf-8.
