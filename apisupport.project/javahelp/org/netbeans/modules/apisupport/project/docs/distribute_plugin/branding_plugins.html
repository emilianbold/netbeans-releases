Branding a Rich-Client Application

See Also

Once the functionality of an application built on top of the NetBeans platform is complete, the appearance of the application still closely resembles the default NetBeans Platform. You can use the IDE to personalize your application. In particular, you can do the following:

  • Replace the launcher's name.
  • Replace the application's splash screen.
  • Define a progress bar.
  • Define the application's title bar.

To brand a rich client application:

  1. Right-click the module suite project in the Projects window and choose Properties.
  2. In the Project Properties dialog box, click Build.
  3. Select the Create Standalone Application checkbox.
  4. In Branding Name, type the name of the launcher.
  5. In Application Title, type the title to be displayed in the application's title bar.
  6. Choose the icon that will be displayed in the application's Help > About dialog box.
  7. Optionally, create a splash screen.
