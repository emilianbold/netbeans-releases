Using the NetBeans Sources and Javadoc

See Also

Once you have registered the NetBeans Sources and Javadoc, you can use them as follows:

  • In the Source Editor, right-click on a class identifier that belongs to the NetBeans APIs and choose Show Javadoc (Alt-F1). Javadoc for the related NetBeans API class is shown in the Source Editor.
  • In the Source Editor, hold down the Ctrl key and move the mouse over a class identifier belonging to the NetBeans APIs. A hyperlink appears. Click on the hyperlink and the cursor jumps to the related NetBeans API class.
