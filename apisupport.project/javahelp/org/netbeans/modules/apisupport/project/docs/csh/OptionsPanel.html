Options Panel Wizard: Location

See Also 

The New Options Panel wizard creates a new Options Panel. You open the New Options Panel wizard by right-clicking a module project, choosing New > Other and selecting Options Panel from the Module Development category.

In the Location page, you have to set the following:

  • Class Name Prefix. Specifies the prefix for the Java source files that the wizard will create.
  • Package. Specifies the name of the package where the Java source files will be housed.
