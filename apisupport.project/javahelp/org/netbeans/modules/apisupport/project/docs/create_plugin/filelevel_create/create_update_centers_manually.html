Manually Registering an Update Center URL

See Also

If you have not been provided with a module that bundles an update center's URL, you need to register the update center manually. Once you have registered the update center, you can access its modules via the Plugin Manager, which you can access under the Tools menu.

To manually register an update center's URL:

  1. Choose Tools > Plugins.

    The Plugin Manager opens.

  2. Click the Settings tab at the top of the Plugin Manager.
  3. Click Add.
  4. Type the name and URL of the autoupdate center. The name can be anything you want it to be. Click Finish.
  5. Click Close.

You have now manually registered the update center's URL.
