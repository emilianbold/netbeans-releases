Building a ZIP Distribution

See Also

Once your rich-client application is complete, one way to distribute it is via a ZIP file. The IDE can create the ZIP file for you, containing the rich-client application.

To build a ZIP distribution:

  1. In the Projects window, right-click the node of the module project you want to build.
  2. Choose Build ZIP Distribution.
