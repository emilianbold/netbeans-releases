New Maven Application Wizard: Module Options

See Also

The New Maven Application wizard enables you to create a new Maven project. You open the wizard by opening the New Project wizard and selecting the project template from the Maven category.

In the Module Options page, you set the following properties:

  • Allow OSGi bundles as Dependencies. When selected, you can use OSGi bundles as dependencies of the NetBeans Platform application or modules.
  • Create Module Project. When selected, the IDE will also create a NetBeans Platform module and configure the module as a dependency of the application or suite. You can specify the name of the module in the text field or use the default name. This option is only available when creating a Maven NetBeans Platform application or module suite.

When you click Finish, the project is created on your computer and opened in both the Projects window and the Files window.
