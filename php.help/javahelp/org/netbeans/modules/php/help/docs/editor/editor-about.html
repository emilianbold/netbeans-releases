The PHP editor provides the following features:

  • Syntax Highlighting
  • Go To Declaration
  • Code Folding
  • Smart Indent
  • Bracket Completion
  • Rename Refactoring and Instant Rename
  • Code Completion
    • Snippets
    • Context-Sensitive Proposals
    • Code Templates and Abbreviations
    • Code Completion in Constructors
    • SQL Code Completion
    • PHP 5.3 Namespaces
    • Overridden and Implemented Methods
  • Code Generators
    • Constructors
    • Getters and Setters
    • Overridden and Implemented Methods
  • Defining Variable Type in Comments
  • Parameter Hints
  • PHP 5.4 Support
  • Error Messages
