Add Correlation Set Dialog Box

See Also

You use the Add Correlation Set dialog box to define correlation sets for the Process element. This dialog box opens when you select the Process element on the diagram and choose Add > Correlation Set in the pop-up menu.

To add a correlation set for the Process:

  1. In the Add Correlation Set dialog box, specify the name for the correlation set and click Add to add properties.
  2. In the Property Chooser dialog box, expand the WSDL file node, select a property to add to the set.
  3. (Optional) Clear the Show Imported Files Only checkbox to view the contents of non-imported WSDL and XML schema files.
    • By default, the Property Chooser dialog box only shows those files that have already been referenced in the process. However, the project may contain other .wsdl and .xsd files which have not yet been imported into the process. If you select a type for the new property that is defined in a non-imported file, the IDE will automatically add the required import to the BPEL process.
  4. Click OK.

To edit the properties of the correlation set:

  • In the Add Correlation Set dialog box, select the correlation set in the list and click Edit.
