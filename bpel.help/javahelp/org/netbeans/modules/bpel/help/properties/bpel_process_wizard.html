New BPEL Process Wizard: Name and Location

See Also

The New BPEL Process wizard enables you to create a BPEL file from scratch. To open the wizard in a BPEL Module project, you right-click the Process Files node and choose New > BPEL Process.

The Name and Location page of the wizard has the following fields:

  • File Name. Specifies the display name of the new BPEL process file.
  • Project Name. (Read-only) Shows the name of the project where you are adding the BPEL file.
  • Folder. Specifies the folder where the IDE will store the BPEL file.
  • Created File. (Read-only) Shows the full path and the name of the file that will be created.
  • Target Namespace. Specifies the namespace of the components defined in the BPEL process file.
