Property Editor Dialog Boxes: Correlations Tab

See Also

You use the Correlations tab to add correlation sets to the Invoke activity manually. This tab is available in the Property Editor dialog box of this Invoke activity.

If you don't want to perform the task manually, use Correlation Wizard.

Before adding correlation to the Invoke element, ensure a correlation set is defined for the Process element.

To add a correlation set to the Invoke element:

  1. On the Correlations tab, click Add.
  2. In the Choose a Correlation Set dialog box, expand the Correlation Sets node, select the correlation set and click OK.
  3. Choose an initiate attribute for this correlation set from the Initiate drop-down list.
  4. Select the pattern for this correlation set from the Pattern drop-down list.
  5. (Optional) Add more correlation sets as needed and click OK.

To edit the properties of the correlation set, select the correlation set in the list and click OK.
