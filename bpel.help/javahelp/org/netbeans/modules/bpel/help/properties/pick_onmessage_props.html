Message Handler Property Editor

See Also

You use this dialog box to configure the properties of On Message branch of the Pick element. The Pick activity is used to wait for one or several possible messages to arrive. The properties of this message that the process awaits from a partner service are configured in the Message Handler Property Editor.

The Message Handler Property Editor dialog box opens when you double-click the On Message branch of the Pick element or click the Property Editor in the Properties window for this element.

  • Partner Link. Select a Partner Link that should send an inbound message.
  • Operation. Select an operation associated with the Partner Link element. The operation is already defined in the partner WSDL file, so you can only select it from the drop-down list.
  • Input Variable. Create a new variable or use an existing variable to hold the received message and transport it for consequent operations.

On the Correlations tab of the Property editor you can manually add correlation sets to the On Message branch. If you don't want to set correlations manually, use the Correlation Wizard.
