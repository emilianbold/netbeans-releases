BPEL Module Project Properties Dialog Box: General

See Also

The General panel allows you to view the path to the folder that contains the project's files, to view or modify the prefix for the project service engine (such as com.sun.bpelse), and to supply aliases and descriptions for the project's Assembly Unit and Application Sub Assembly.

You open the Project Properties dialog box by right-clicking a BPEL Module project and choosing Properties.
