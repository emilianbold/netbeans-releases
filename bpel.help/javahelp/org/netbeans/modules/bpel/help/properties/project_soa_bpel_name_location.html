New BPEL Module Wizard: Name and Location

The New Project wizard steps you through the process of creating a new project. You open the wizard by choosing File > New Project and selecting the BPEL Module project template from the SOA category.

The Name and Location page of the wizard prompts you to supply information characterizing the BPEL Module project you are creating:

  • Project Name. Supply a name for the folder that will contain your BPEL Module files.
  • Project Location. Keep the default path to your other projects, or specify a new one.
  • Project Folder (not settable). The system derives this value by concatenating the previous two entries.
  • Set as Main Project. Keep this check box selected unless you want a different project to be regarded as the main project.

When you click Finish, the project is created on your computer and opened in both the Projects window and the Files window.
