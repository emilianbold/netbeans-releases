Adding Folders to BPEL Module Projects

See Also

All files related to your BPEL process are stored in folders, which are directories on your hard drive. The logical structure of project folders is displayed in the Projects window. A directory-based structure of project folders is displayed in the Files window.

To add a folder to a project:

  1. In the Projects window, expand the BPEL Module project node, right-click the default Process Files node, and choose New > File/Folder from the pop-up menu.
  2. In the New File dialog box, under Categories, select the Other node. Then, in the File Types list, select Folder and click Next.
  3. In the Name and Location page, name the new folder and specify its location in the parent src folder within the project directory structure (you can see the project's directory structure in the Files window).
  4. Click Finish.
