Navigating in the BPEL Design View

See Also

When working with large-scale diagrams, use the navigation features provided in the Design view.

Using the Thumbnail View

The Thumbnail view helps you clickly navigate to another portion of your business diagram. This view provides a minuature representation of your entire diagram. The current display area is shown with a shaded rectangle. The Thumbnail view can be activated only if your diagram exceeds the size of the Design view area.

To navigate to a portion of the diagram using the Thmbnail view:

  1. Open a BPEL file in the Design view.
  2. In the right lower corner of the Design view, click the Thumbnail view icon .
    Note that this icon is visible only when the diagram exceeds the size of the Design view area.
  3. Drag the shaded rectangle to the place on the diagram that you need.

Zooming In and Out

The zoom feature allows you to reduce or enlarge the size of your diagram to get a closer view or to see more of the diagram at a reduced size.

To change the current zoom of a diagram:

  • On the Editor toolbar, do one of the following:
    • Move the zoom slider until you achieve the desired magnification.
    • Click Fit Diagram to scale the diagram to fit the window.
    • Click Fit Width to scale the diagram width to fit the window.
  • Note that the minimum scale size is 33% and some large diagrams might not fit entirely the window.

Using the Navigation Mode

Use the Navigation Mode to scroll your diagram in any direction. This feature lets you navigate around large-size diagrams.

To adjust the position of the diagram:

  • On the Editor toolbar, click the Navigation Mode icon and drag the diagram in the direction you need.
    To return to editing the diagram, click the Editing Mode icon .

Scrolling through the Diagram in Editing Mode

When you open a BPEL file from the Projects window, the diagram opens in the Editing Mode of the Design view by default. In this mode, you can edit the diagram and scroll through it. The Editing Mode is enabled when the icon is selected on the Editor toolbar.

In the Editing Mode, you can scroll through the diagram by using the following methods:

  • Turning the mouse wheel
  • Using the horizontal and vertical scroll bars
  • Pressing the Tab key or Shift-arrow to move through elements

Using the Navigator window

You can navigate to an element on the diagram by selecting that element in the Navigator window. Conversely, when you select an element on the diagram, that element becomes selected in the Navigator window.

To navigate to an element on the diagram using the Navigator window:

  1. Open a BPEL file in the Design view.
  2. In the Navigator window, select BPEL Logical View from the drop-down list.
  3. In the Navigator tree, select an element you want to find on the diagram.
    This element is selected on the diagram in the Design view.
