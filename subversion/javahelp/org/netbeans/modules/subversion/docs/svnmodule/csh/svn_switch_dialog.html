Subversion: Switch Dialog Box

See Also

The Subversion Switch dialog box enables you to update your working copy to point to a new location the repository.

You access this dialog box by selecting a versioned file or folder (e.g. from the Projects, Files, or Favorites window) and choosing either Subversion > Switch to Copy from the right-click menu or Versioning > Switch to from the IDE's main menu.

You can specify a new location by entering a repository folder and optionally, a revision number:

  • Repository Folder. Enter the path to a location in the repository. Alternatively, click the Browse button and select a folder from the interface.
  • Repository Revision. If you need to specify a previous revision, enter the revision number for the selected location. Leave blank if you require the most recent revision. To display a log of all revisions for the selected location, you can click the Search button.
  • When you click Switch, the location of your working copy takes on the content reflected by the newly selected location in the repository. Note that any differences arising between your local working copy and the location that you are switching to will be overwritten by the new location.
