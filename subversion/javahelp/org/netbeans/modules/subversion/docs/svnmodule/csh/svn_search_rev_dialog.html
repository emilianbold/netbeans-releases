Subversion: Search Revisions Dialog Box

See Also

The Subversion Search Revisions dialog box enables you to specify a revision for the repository folder you previously select when checking out from the repository.

The Search Revisions dialog box is accessed by clicking the Search button next to the Repository Revision field in the Subversion Checkout wizard.

To search for a revision:

  1. Enter a date in the log entries field (YYYY-MM-DD format) to set a beginning point for the search.
  2. Click the List button to initiate the search. All revisions based on the search criteria display in the text area.
  3. Select a displayed revision from the text area, then click OK. When you click OK, the selected revision appears in the Repository Revision field of the Checkout wizard.
  • In order to perform a search on all revisions, leave the log entries field blank.
