Subversion: Search Revisions Dialog Box

See Also

The Subversion Search Revisions dialog box enables you to specify a revision for the repository folder you previously select when examining data from the repository.

This dialog box is accessed by clicking the Search button next to the Repository Revision field in various wizards and dialogs.

To search for a revision:

  1. Enter a date in the log entries field (YYYY-MM-DD format) to set a beginning point for the search. Leave this field empty to select all available revisions.
  2. Click the List button to initiate the search. All revisions based on the search criteria display in the text area.
  3. Select a displayed revision from the text area, then click OK. When you click OK, the selected revision appears in the Repository Revision field of the wizard or dialog you are working from.
  • In order to perform a search on all revisions, leave the log entries field blank.
