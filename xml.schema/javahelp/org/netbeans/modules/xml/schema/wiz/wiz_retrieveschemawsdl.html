New Retrieve Documents Wizard: Specify Resource Location

See Also

The Retrieve Documents wizard lets you retrieve an external XML schema document or an external WSDL (Web Services Description Language) document into your project. The retrieval of resources retrieves both the documents and their dependencies. The wizard verifies that the transitive closure is satisfied, including XML schema embedded in WSDL. You open the Retrieve Documents wizard by right-clicking the project node of a web application project or a Java application project, choosing New > File/Folder then selecting External XML Schema Document or External WSDL Document in the XML category.

The Specify Resource Location page has the following fields:

  • From URL. Specifies the URL of the external document.
  • From Local File System. Specifies the folder where the external schema files reside.
  • Save to Folder. Specifies the folder where you want to store the retrieved files.
  • Overwrite files with same name. Specifies whether the retrieval should overwrite files with the same name in your project.
