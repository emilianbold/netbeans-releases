New XML Schema Wizard: Name and Location

See Also

The New XML Schema wizard lets you create a new XML schema. You open the New XML Schema wizard by right-clicking the Source Packages node of a web application project or a Java application project, choosing New > File/Folder and selecting XML Schema in the XML category.

The Name and Location page has the following fields:

  • File Name. Specifies the display name of the new XML schema file.
  • Project. (Read-only) Shows the name of the project where you are adding the schema file.
  • Folder. Specifies the folder where the IDE will store the XML schema file.
  • Created File. (Read-only) Shows the full path and the name of the file that will be created.
  • Target Namespace. Specifies the namespace of the components defined in the XML schema file.
