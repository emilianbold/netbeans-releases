Adding an annotation Component to an XML Schema File

See Also

You use an annotation to enclose documentation components which specify schema comments. XML schema tools let you work with annotation components and documentation sub components.

To add an annotation component:

  1. In the IDE, open the schema file where you want to add the annotation component.
  2. In the Schema view, right-click the node of the schema component where you want to add the annotation component and choose Add > Annotation or Add Annotation from the pop-up menu.
    The IDE adds an annotation node.
