Adding an attributeGroup Component to an XML Schema File

See Also

You use an attributeGroup to define an attribute group to use in complex type definitions.

To add an attributeGroup component:

  1. In the IDE, open the schema file where you want to add the attributeGroup component.
  2. In the Schema view, right-click the root node of the schema and choose Add > Attribute Group from the pop-up menu.
    -or-
    In the Schema view, right-click the Attribute Groups node and choose Add Attribute Group from the pop-up menu.
    The IDE adds a node for the new attribute group under the Attribute Groups node.
