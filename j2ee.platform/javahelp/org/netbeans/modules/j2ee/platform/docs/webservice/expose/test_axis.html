Testing Axis2 Web Services

See Also

To test a deployed Axis2 web service in the IDE, right-click on one of the web service's subnodes and select Test Operation in Browser. A browser window opens with XML output and test values for variables. These test values are appended to the end of the URL in the browser window. Change the values in the URL and press Enter, and the output reloads with the new values.
