Working with the JBoss Application Server

The following table shows how to work with the JBoss Application Server.

Set up the JBoss Application Server
  1. Register the JBoss Application Server with the IDE.
  2. Optionally, set up a connection pool

Deploy to the JBoss Application Server
  1. Set the JBoss Application Server as the project's target server.
  2. Do one of the following:
    • To start the JBoss Application Server automatically, deploy the web application, web service, web service client, EJB module, or enterprise application.
    • To start the JBoss Application Server manually, right-click the server instance in the Services window and choose Start or Start in Debug Mode.
