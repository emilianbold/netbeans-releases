Working with Web Services

The following table shows how to work with web services.

Design the web service
  1. Go to Help  BluePrints Solutions Catalog.
  2. Read topics under the "Service Oriented Architectures with Web Services" item.

Create the web service
  1. Prepare to develop web services.
  2. Do one of the following:
    • Implement an existing WSDL file
    • Publish existing code as a web service
    • Create a web service from scratch

Develop the web service
  1. In the Projects window or Files window, double-click the files that you would like to edit.
  2. Use the Source Editor to develop the web service. When you right-click in the Source Editor, you can use the IDE to generate skeleton code for one or more of the following:
    • Add a web service operation
    • Add a SOAP message handler
    • Call an EJB

Build the web service
  • Do one of the following:
    • Build a web service from a WSDL file
    • Build a document-style web service
    • Build a RPC-style web service

Publish the web service

You can publish a web service as a stand-alone module or as a part of a J2EE application.

  1. To publish it as part of a J2EE application, right-click the J2EE application's J2EE Modules node, choose Add J2EE Module, select the module that contains the web service, and click OK.
  2. Make sure the appropriate server is targeted.
  3. Choose Run > Run Main Project or right-click the project and choose Run Project.

Test the web service
  1. Register the web service.
  2. Test the web service from the IDE.

Consume the web service
  1. Create the web service client.
  2. Test the web service from the IDE.
  3. Once you are sure that the web service is useful to you, use the Source Editor to develop the web service client. When you right-click in the Source Editor, you can use the IDE to generate skeleton code for one or more of the following:
    • Call a web service operation.
    • Add a SOAP message handler
    • Call an EJB
  4. Build the web service client
  5. Make sure the appropriate server is targeted.
  6. Choose Run > Run Main Project or right-click the project and choose Run Project.
