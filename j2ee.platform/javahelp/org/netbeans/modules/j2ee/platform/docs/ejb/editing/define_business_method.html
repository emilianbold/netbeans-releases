Defining a Business Method

See Also

A business method is a method inside an enterprise bean that performs some function and is available for any clients of the bean. It can be declaratively secured or configured to follow some transaction semantics by configuring the ejb-jar.xml deployment descriptor.

To define a business method for an enterprise bean:

  1. In the Projects window, expand the Enterprise Beans node and the node for the EJB.
  2. Right-click enterprise bean's node and choose Add  Add Business Method.

    You can also open the bean class and right-click anywhere inside the body of the class and choose EJB Methods  Add Business Method. This command is only enabled when you click within the body of the class (after the class declaration).

  3. In the Add Business Method dialog box, define the method signature:
    1. Name the method.
    2. Specify the return type.
    3. Specify the method parameters in the Parameters tab.
    4. Specify the exceptions that the method throws in the Exceptions tab.
  4. Click OK to create the method.

    The IDE adds the method signature to the remote interface and the method body to the bean class.

  5. Edit your new business method.

Notes:

  • When you create an enterprise bean in the IDE, the IDE creates a special BeanNameRemoteBusiness interface in which it registers business methods. The BeanNameRemote interface extends the BeanNameRemoteBusiness and the bean class implements both the BeanNameRemote interface and the BeanNameRemoteBusiness interface.
  • When the IDE adds a business method to an enterprise, it does not save the bean class or the remote interface. Only when you compile or save the class is the new method saved to disk.
  • To fully remove a business method from both a enterprise bean's bean class and remote interface or remote business interface, delete the method from the bean's Home Methods list in the Projects window. If you delete the business method in the Source Editor, the IDE does not remove it from the remote interface or remote business interface.
