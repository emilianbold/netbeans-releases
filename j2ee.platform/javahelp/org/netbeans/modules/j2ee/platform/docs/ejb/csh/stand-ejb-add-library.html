Add Library Dialog Box

See Also 

You open this dialog box by right-clicking the Libraries node of an EJB module or web module and choosing Add Library. The Add Library dialog box lets you choose the libraries that the module or application contains on its classpath. The Add Library dialog box list the libraries registered with the Library Manager.

To add a library, click Manage Libraries to open the Library Manager. Alternatively, choose Tools > Libraries from the main menu to open the Library Manager.
