Viewing the Published WSDL File

See Also 

After successfully deploying a web service, the container published the WSDL file. You might want to see the published WSDL file as evidence of a successfully deployed web service.

To view a published WSDL file:

  1. Open a browser.
  2. Go to this URL: 
    context_path/web_service_nameService?WSDL

    The WSDL file opens in the browser.
