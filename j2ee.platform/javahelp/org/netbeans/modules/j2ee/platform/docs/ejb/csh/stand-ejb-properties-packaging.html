Standard EJB Module Project Properties Dialog Box: Packaging

See Also 

You open the Standard EJB Module Project Properties dialog box by right-clicking any  and choosing Properties. The Packaging page lets you set up the filter for which files are included in the EJB JAR file.

Notes:

  • To change the JAR name and location, expand the project's folder in the Files window and open project.properties. Enter the full path to the JAR file folder in the dist.dir property and the JAR file's name in the jar.name property.
  • To manage the project's classpath, click Libraries in the left panel of the dialog box. To set command-line options for javac, click Compiling.
