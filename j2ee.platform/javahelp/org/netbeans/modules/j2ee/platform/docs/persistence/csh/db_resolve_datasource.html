Resolve Data Sources Dialog Box

See Also

This dialog box is displayed when you right-click a project that has a reference problem and choose Resolve Data Source Problem. This problem can appear when the project specifies the details of a data source that the IDE cannot resolve because no corresponding connection is registered with the IDE. The dialog displays the name of the data source that is specified in the project and that needs to be resolved.

To resolve the problem, click Add Connection to open the New Connection Wizard and create a database connection for the data source.
