Add Server Instance Wizard: GlassFish 3 Server Location

See Also

You see this page of the Add Server Instance wizard when you specify details to register an instance of GlassFish 3 with the IDE. To register a server instance, you specify the properties of a local or remote domain.

Use this page to specify the following information about the location of GlassFish 3.

  • Installation Location. Supply the path to a local installation of GlassFish 3 server. Click Browse to navigate to the location of the local installation of the server.

    If there is no installation of GlassFish 3 on your local system, you can download and install the GlassFish 3 server from the wizard by clicking the Download button. After you click Download, the IDE will install the current version of GlassFish 3.

    • You must accept the terms of the licensing agreement to enable the Download button.

After you specify the location of the GlassFish 3 installation, click Next to specify the details of the local or remote domain.
