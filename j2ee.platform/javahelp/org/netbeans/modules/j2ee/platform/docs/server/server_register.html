Registering a Server

See Also

Before you can deploy an enterprise application, web application, JSP file, servlet, or EJB module, the server to which you are going to deploy needs to be registered with the IDE.

To register an external server:

  1. Choose Tools > Servers in the main menu.
  2. In the Server Manager, click Add Server.

    3. The Add Server wizard appears and displays the types of servers that are compatible with the IDE.

  3. In the dialog box, select the type of server you want to register and click Next.
  4. Specify the server-specific information in the panels that follow and click Finish.
  • Alternatively, you can open the Add Server wizard by right-clicking the Servers node in the Services window and choosing Add Server.
