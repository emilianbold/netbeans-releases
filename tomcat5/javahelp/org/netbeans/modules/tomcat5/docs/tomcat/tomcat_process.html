Working with the Tomcat Web Server

The following table shows how to work with the Tomcat Web Server.

Set up the Tomcat Web Server
  1. Register the Tomcat Web Server with the IDE.
  2. View the Tomcat Web Server in the IDE
  3. Do one or more of the following:
    • Customize the Tomcat Web Server
    • Set up a connection pool
    • Write Ant targets for deployment to Tomcat Web Server 4
    • Use the Java Web Services Developer Pack 1.6
    • Write Ant Targets for web service deployment and consumption

Deploy to the Tomcat Web Server
  1. Set the Tomcat Web Server as the project's target server. For versions of the IDE that are bundled only with the Tomcat Web Server, the Tomcat Web Server is the target server by default.
  2. Do one of the following:
    • To start the Tomcat Web Server automatically, deploy the web application.
    • To start the Tomcat Web Server manually, right-click the server instance in the Services window and choose Start or Start in Debug Mode. Start the server manually when you, for example, use a debug target in a free-form project .
