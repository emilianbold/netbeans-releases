About IEP Data Types

See Also

IEP supports eight data types for use as attributes of IEP operators, as specified in the property editors for the IEP operators:

  • BIT
  • INTEGER
  • BIGINT
  • DOUBLE
  • VARCHAR
  • DATE
  • TIME
  • TIMESTAMP
