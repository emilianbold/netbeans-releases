Using the Time Based Window Operator Property Editor

See Also

The Time Based Window operator retains all records within the specified period of time; that is, any event whose timestamp falls in the range specified by the current time minus the specified period of time.

You can combine the Time Based operator with a subsequent Relation Output operator to provide a timestamp record for the records retained.

To create a Time Based Window operator:

  1. Drag a Time Based Window operator to the Design view.
  2. Click the Time Based Window operator icon.The property editor opens.
  3. In the Size field, type the amount of time to retain records.
  4. In the drop-down menu, specify the units of time.
  5. Click Ok.
