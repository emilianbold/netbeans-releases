Using the Table Input Operator Property Editor

See Also

The Table Input Operator provides static event information to the IEP feature by providing a relational table as input. You can use the operator as table input to other downstream operators.

To create a Table Input operator:

  1. Drag a Table Input operator from the Palette to the Design view.
  2. Click the Table Input operator icon.The property editor opens.
  3. The Details section of the property editor displays the name of the operator and the output schema.
  4. In the Attributes section, you can add or delete attributes for this table input, or move the attributes up or down to reorder them.
  5. In the Data Type column in the Attributes section, select the appropriate data type from the list of data types.
  6. In the Size column, you can specify the size of the data type if, for example, you specify the VARCHAR data type.
  7. Click OK.
