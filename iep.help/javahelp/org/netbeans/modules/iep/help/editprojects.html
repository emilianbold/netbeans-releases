Editing Intelligent Event Processing Module Projects

See Also

You can edit Intelligent Event Project modules you develop.

To edit an IEP project node using a Property Editor:

  1. After dragging an operator to the Design view, click on the operator now in the Design view. The Property Window opens.
  2. In the Property window, make your changes.
  3. Click OK.
  4. Once you have created and connected all operators for your IEP process, you can click the validation icon at the top of the IEP Design view to validate the operators that make up your IEP process.
    The validation process runs a set of predefined validation rules, and provides output in the IDE's output area. For example, one validation rule checks to see if all required operator properties have been set, and reports in the output area if the property has not been set.
