Using the Table Operator Property Editor

See Also

The Table operator provides a snapshot of the current state of a relation.

To create a Table operator:

  1. Drag a Table operator from the Palette to the Design view. You cannot change any attributes in the Table Operator but you can examine them.
  2. Click the Table operator icon. The property editor opens.
  3. The Details section of the property editor displays the name of the operator and the output schema.
  4. The Attributes section provides a picture of the current state of the input relation.
  5. Click OK.
