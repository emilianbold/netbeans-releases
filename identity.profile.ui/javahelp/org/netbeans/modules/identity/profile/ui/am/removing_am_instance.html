Removing Sun Java System Access Manager Instances

See Also

When you no longer need a Sun Java System Access Manager instance, you can use the Services window to remove the instance from the IDE.

To remove a Sun Java System Access Manager instance:

  1. In the Services window, expand the the Sun Java System Access Managers node.
  2. Right-click the server-instance node you want to remove and choose Remove.
  3. Click Yes to confirm.
