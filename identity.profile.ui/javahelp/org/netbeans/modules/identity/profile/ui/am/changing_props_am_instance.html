Changing the Properties of Sun Java System Access Manager Instances

See Also

You can modify the properties of the Sun Java System Access Manager instance after you have added it in the Services window.

To change the properties of an instance of the Sun Java System Access Manager:

  1. In the Services window, right-click the Sun Java System Access Managers node and choose Properties.
    The Edit server-instance dialog box opens.
  2. Make your changes and click OK.
