About the SQL Pane

See Also

The SQL pane, the third pane from the top in the Visual SQL Editor, displays the text of the SQL query.

Some things you can do with queries in the SQL pane:

  • Enter and edit SQL code. If you make changes in the SQL pane, the diagram pane and grid pane are updated after you select Parse Query.
  • Test the SQL query by right-clicking and choosing Run Query. A representation of the results appear in the results pane.
  • Parse the SQL query by right-clicking and choosing Parse Query.
  • If the SQL pane contains code that is not supported by the diagram pane or the grid pane, the diagram and grid panes become blank, but you might still be able to execute the query.
