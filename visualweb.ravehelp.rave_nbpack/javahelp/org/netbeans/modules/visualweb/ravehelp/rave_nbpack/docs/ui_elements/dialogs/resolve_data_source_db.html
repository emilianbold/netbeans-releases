Resolve Data Source(s) Dialog Box

See Also

The Resolve Data Source(s) dialog box helps you resolve database problems when you import an existing Java Studio Creator 2 project into the IDE. You open this dialog box by expanding the project-name node in the Projects window, right-clicking the Data Source References node, and choosing Resolve Data Source(s).

The Resolve Data Source(s) dialog box provides you with a list of JNDI names. Choose the appropriate name and click Add Data Connection to continue to the Edit DataBase Server Types dialog box.
