Adding Components in the Page Flow Editor

See Also | Tutorials

You can add components involved with navigation (hyperlinks, image hyperlinks, and buttons) to a page in the Page Flow editor. Adding components in the Page Flow editor is faster than returning to the Visual Designer and adding components there. If you have added third party components to the Palette that you want to use for navigation, you must add those components in the Visual Designer.

To add a navigation component to a page:

  • In the Page Flow Editor, right-click the page and choose Add Hyperlink, Add Image Hyperlink, or Add Button.
