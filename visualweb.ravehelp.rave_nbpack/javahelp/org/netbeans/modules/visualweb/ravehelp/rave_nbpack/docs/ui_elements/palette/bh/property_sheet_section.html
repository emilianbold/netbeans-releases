Property Sheet Section Component

See Also | Tutorials

You can drag the Property Sheet Section component  from the Palette's Layout category to a property sheet on the Visual Designer to add a section to a Property Sheet, or you can right-click a Property Sheet in either the Visual Designer or the Navigator window and choose Add Property Sheet Section. A Property Sheet Section contains Property components and is used primarily as a way to group Property components in a Property Sheet and to provide a label for the group.

  • This component is a JavaServer Faces 1.1 component. You see components that use this version of JavaServer Faces when your project uses J2EE 1.4 or J2EE 1.3.

You can set the text that appears for a Property Sheet Section by editing its label component in the Property Sheet Section Properties Window.

A property sheet section contains Property components. A new Property Sheet Section intially has one Property component in it.

To add a property to a section, either drag a Property component to a Property Sheet Section and drop it or right-click the Property Sheet Section component and choose Add Property.
