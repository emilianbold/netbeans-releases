JSF Standard Components Category

See Also | Tutorials

This category of the Palette contains the old style JavaServerTM Faces visual components. These components are reference implementations of the JavaServer Faces API that do not support themes and in most cases are more complex to work with than the components in the Basic, Layout, and Composite categories of the Palette. To get advanced functionality and the enhanced look and feel provided by themes, use the the Basic, Layout, and Composite components instead.

You can drag the Standard components to the Visual Designer and use them in designing your pages. In the Palette, you can select a component and press F1 to see a help topic for that component.

After you add a component to your page, you can see it listed in the Navigator window, which is typically located below the Palette. The Navigator window is a useful tool for selecting components, especially components that are hard to select in the Visual Designer.
