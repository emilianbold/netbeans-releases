Adding a Web Service to the IDE

See Also

Before you work with a web service in the IDE, add it to the Servers window. To add a web service, specify the location of the Web Service Definition Language (WSDL) file. A WSDL file might contain one or more web services. If you add a WSDL file with multiple services to the IDE, all services referenced in the WSDL file are available to the IDE.

By default, several web services are installed with the IDE.

How you add a web service depends on whether the web service source is stored in a local file or accessed via a URL.

To add a web service from a local file:

  1. In the Servers window, right-click the Web Services node or the group node and choose Add Web Service.
  2. In the Add Web Service dialog box, select Local file and click Browse.
  3. In the Open dialog box, browse to the WSDL file, select the file, and click Open.

    For example, you could select the JavaTravellerService.wsdl file shipped with the IDE.

  4. In the Add Web Service dialog box, view the web service information to ensure that it is correct.
  5. If desired, test the web service's methods by completing the following steps:
    1. In the Web Service Information list, click Test Method.
    2. Type a test value for the parameters and click Submit.
    3. View the results and click Close.
  6. In the Add Web Service dialog box, click Add.

    After you add a web service, its node appears as a subnode of the Web Services node or a group node in the Servers window.

To add a web service from a URL:

  1. In the Servers window, right-click the Web Services node or the group node and choose Add Web Service.
  2. In the Add Web Service dialog box, select URL.
  3. If you are behind a firewall, set your HTTP proxy.
    • If you are using web services deployed to servers on localhost (your own computer), you might want to modify the browser's settings to specify not to use a proxy server:
      • Internet Explorer: From the main menu, choose Tools > Internet Options.
        Click the Connections tab, the LAN Settings button, and the Advanced button.
        Add localhost to the Do not use proxy list.
      • Netscape: From the main menu, choose Edit > Preferences. Open the Advanced node and select Proxies. Click View. Enter localhost in the Do not use proxy list.
      • Mozilla: From the main menu, choose Edit > Preferences. Open the Advanced node and select Proxies. Enter localhost in the No proxy for list.
  4. In the Add Web Service dialog box, type the URL.
  5. Click Get Web Service Information to view the web service information to ensure that it is correct.
  6. If desired, test the web service's methods by completing the following steps:
    1. In the Web Service Information list, click Test Method.
    2. Type a test value for the parameters and click Submit.
    3. View the results and click Close.
  7. Click Add.

    After you add a web service, its node appears in the Web Services node in the Servers window.
