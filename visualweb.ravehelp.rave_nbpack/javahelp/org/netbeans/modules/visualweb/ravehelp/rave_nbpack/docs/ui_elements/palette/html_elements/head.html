head Component

See Also | Tutorials

The head component is created for you when you create a new page. You can see this component in the Navigator window when you select a page in the Visual Designer in Design mode and open the html node for the page.

On the rendered HTML page, this component produces a <head> tag, which typically contains information about the page. A typical tag in a Visual Web Application page's <head> section is the <link> tag that indicates page's style sheet.

The JSP representation of this tag is bound to the current page. For example, if the page is named Page1.jsp and the server is a Java EE 5 server, the default JSP code for the head component is as follows:

   <webuijsf:head binding="#{Page1.head1}" id="head1">
   . . . (tags such as <link>, <meta>, and <title>)
   </webuijsf:head>
