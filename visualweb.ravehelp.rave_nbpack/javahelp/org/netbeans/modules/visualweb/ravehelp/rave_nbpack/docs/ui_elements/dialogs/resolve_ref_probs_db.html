Resolve Reference Problems Dialog Box

See Also

The Resolve Reference problems dialog box helps you resolve reference problems when you import an existing Java Studio Creator 2 or NetBeans Visual Web project into the IDE. You open this dialog box by right-clicking the badged project-name node in the Projects window and choosing Resolve Reference Problems.

The Resolve Data Source(s) dialog box provides you with a list of missing references to be resolved. Select the problem and click Resolve to be taken to the Library Manager.
