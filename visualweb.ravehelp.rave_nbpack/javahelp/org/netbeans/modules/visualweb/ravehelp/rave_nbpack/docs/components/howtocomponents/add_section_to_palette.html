Adding a Category to the Palette

See Also

The , typically located in the upper left portion of the IDE, provides components and code clips that you can use in the currently selected page or code editor. There are two views of the Palette, Components and Code Clips, that display depending on the type of editor you are using. You see components when you are in the Visual Designer, and you see code clips when you are in a source code editor. Each view has categories that group its contents. You can add categories to either view, and you can delete categories that you have added.

To add a category to the Palette:

  1. Open the editor that is appropriate for the view you want to see. For example, with the Visual Designer open, you see the component categories. If you click the JSP or Java button in the Visual Designer toolbar, you see code clip categories.
  2. If necessary, open the Palette by choosing Window > Palette.
  3. Right-click any category (for example, Basic in the Components view) and choose Create New Category.
  4. In the New Palette Category dialog box, enter a name for the category.
  5. Click OK.

    The new category appears in the Palette.
