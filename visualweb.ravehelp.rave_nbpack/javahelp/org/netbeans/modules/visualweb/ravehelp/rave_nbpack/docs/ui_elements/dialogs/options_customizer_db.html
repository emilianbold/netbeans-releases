Options Customizer Dialog Box

See Also

This dialog box enables you to customize parameters for list components. Right-click the component in the Visual Designer to open the Options Customizer dialog box.

You can customize the following options:

  • Select items. Specify which items are selected by default.
  • Allow multiple selections. If the Select items checkbox is selected, you can specify multiple items that are selected by default.
    • Note: This option is available in components that support more than one selection.
  • Display. Specify the text that appears in the component.
  • Value. Specify the value of the selection.
Click New to add new items to the list of items. Select the item and click Up or Down to change the position of items on the list.
