Naming a JNDI Data Source in a Visual Web JSF Page

See Also | Tutorials

In the Visual Web framework, you can name a data source that you drag and drop onto a Visual Web JSF page. This can be very useful when you work with more than one data source.

To enable this feature, do the following:

  1. Choose Tools > Options.
    The Options Window opens.
  2. Choose Miscellaneous > Visual Web.
  3. Check the Prompt for Data Source Name checkbox.

After the feature has been enabled, whenever you drag and drop a data source onto a Visual Web JSF page, the specify JNDI Name dialog box opens. Enter the name of the data source and click OK.
