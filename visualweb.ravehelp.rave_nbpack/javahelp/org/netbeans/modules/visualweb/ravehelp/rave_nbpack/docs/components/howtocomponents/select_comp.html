Selecting Components in a Page

See Also | Tutorials

You can select components in several different ways.

To select a single component:

  • In the or the click the component.

To select multiple components, use any of the following techniques:

  • In the or the click each component while holding down the Control or Shift key.
  • In the Outline window, click the first item in the list, then hold down the shift key and click the last item in the list. All components in between are selected.
  • In the Visual Designer, click and hold the mouse button down, then drag the mouse to enclose fully all the items to select.
