Run Query Window

Use this window to define an SQL query to view the data in a database to test queries or to test operations performed by your application on a database. To open this window, in the , expand the Databases node and the database-name node, right-click a table, and choose View Data.

  • To define and save queries that will be run as part of your application, use the Visual SQL Editor.

Some things you can do in this window:

  • Create a new query by typing it in the Query text field.
  • Modify an existing query in the Query text field.
  • Run the query by right-clicking in the Query text field and choosing Run Statement.

Within a given window, the database connection is established when a query is first run. The connection stays open until you close the window. Each Run Query window has its own database connection.
