Renaming a Page Navigation Connector

See Also | Tutorials

You can rename a connector in the Page Navigation Editor or by using the connector's Properties window.

To rename a connector in the Page Navigation Editor:

  1. In the Page Navigation Editor, double-click the connector you want to rename.
  2. Select the text, type in the new name, and press Enter.

To rename a connector by using the Properties window:

  1. In the Page Navigation Editor, select the connector to rename.
  2. In the Properties window, select the text value to the right of the Outcome property.
  3. Type in the new name and press Enter.
