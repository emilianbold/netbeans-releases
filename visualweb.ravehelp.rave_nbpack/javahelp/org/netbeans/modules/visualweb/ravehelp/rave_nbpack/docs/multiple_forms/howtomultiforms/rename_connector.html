Renaming a Page Flow Connector

See Also | Tutorials

You can rename a connector in the Page Flow editor or by using the connector's Properties window.

To rename a connector in the Page Flow editor:

  1. In the Page Flow editor, right-click the connector you want to rename.
  2. Type the new name and press OK.

To rename a connector by using the Properties window:

  1. In the Page Flow editor, select the connector to rename.
  2. In the Properties window, select the text value to the right of the Outcome property.
  3. Type in the new name and press Enter.
