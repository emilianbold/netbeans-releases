Deleting a Web Service

See Also

If you do not want to use a web service on a particular page, you can delete it from the page. You can also delete web services from the Servers window. Once you delete a web service from the Servers window, you can no longer use that web service in any page.

To delete a web service from the page:

  • In the Outline window, right-click the web service and choose Delete.

To delete a web service from the Servers window:

  • In the Servers window, right-click the web service and choose Delete.
