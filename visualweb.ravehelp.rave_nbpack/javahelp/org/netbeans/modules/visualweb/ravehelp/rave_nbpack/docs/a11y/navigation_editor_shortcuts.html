Page Navigation Editor Keyboard Shortcuts

See Also

This topic describes how to use the keyboard shortcuts to work with the Page Navigation Editor in the Visual Web Pack.

To perform this task... Follow these steps...
Open the Page Navigation Editor

If the focus is in the Design view of the Visual Designer:

  1. Press Shift-F10.
  2. Press the down arrow to scroll to the Page Navigation selection, and then press Enter.

(Optional) If the focus is in the Design view of the Visual Designer:

  1. Press Ctrl-1. The focus changes to the Projects window.
  2. Press the right arrow to open a project node.
  3. Press the down arrow and scroll to the Page Navigation selection, and then press Enter.
Change the focus to the open Page Navigation editor

If the Page Navigation editor is open:

  • Press Ctrl-Tab to change the focus to the Page Navigation editor.
See the page view options

With the focus on the Page Navigation Editor:

  • Press Z or z. The selected page view becomes larger.
  • Press U or u. The selected page view becomes smaller.
  • Press Tab. The focus changes to the next page.

Note that the selected page has a blue border.
Switch between page views With the focus on a page in the Page Navigation Editor:
  • Press Tab. The focus changes to the next page.
Add a page

With the focus on the Page Navigation editor:

  • Press Ctrl-A to add a page.
Delete a page

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page that you want to delete.
  2. Press Delete to delete the selected page.
Add a button to a page

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page to which you want to add the button.
  2. Press Z to enlarge the page view.
  3. Press B to add a button. The button is added to the page.
Add a hyperlink to a page

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page to which you want to add the hyperlink.
  2. Press Z to enlarge the page view.
  3. Press L to add a hyperlink. The hyperlink is added to the page.
Add an image hyperlink to a page

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page to which you want to add an image hyperlink.
  2. Press Z to enlarge the page view.
  3. Press I to add an image hyperlink. The image hyperlink is added to the page.
Open selected page in the Design window

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page that you want to open.
  2. Press Enter. The page opens in the Design window.
Create a navigation link

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page that you want to use as the start.
  2. Press S. The selected page outline color changes to red.
  3. Press Tab to select the page to which you want to link.
  4. Press E to end the link. The linked pages outline colors change to red.
  5. Edit the link name.
  6. Press Enter. The start page outline color returns to black.
Create a navigation link for a button

With the focus on the Page Navigation editor:

  1. Press Tab to change the focus to the page that you want to use as the start.
  2. Press Z. The selected page view becomes larger.
  3. Press the up arrow or down arrow keys to select the button from which you want to link.
  4. Press S to mark the page and button as the start of the link.
  5. Press Tab to change the focus to the page to which you want to link the button.
  6. Press E to end the link. The linked pages outline colors change to red.
  7. Edit the link name.
  8. Press Enter. The start page outline color returns to black.
Move a page

With the focus on the Page Navigation editor:

  1. Press Ctrl-D to change to the draggable mode.
  2. Press Tab to select the page you want to move.
  3. Press the up arrow, down arrow, right arrow, or left arrow keys to move the selected page.
Set the layout mode to fixed

With the focus on the Page Navigation editor:

  • Press Ctrl-D to change to the fixed mode.
Change the appearance of the lines

With the focus on the Page Navigation editor:

  • Press Ctrl-S to change to straight lines.
  • Press Ctrl-W to change to wired lines.
Close the Page Navigation editor
  • Press Ctrl-F4
