Page Flow Editor Keyboard Shortcuts

See Also

This topic describes how to use the keyboard shortcuts to work with the Page Flow Editor.

To perform this task... Follow these steps...
Open the Page Flow Editor

If the focus is in the Design view of the Visual Designer:

  1. Press Shift-F10.
  2. Press the down arrow to scroll to the Page Navigation selection, and then press Enter.
Change the focus to the open Page Flow editor

If the Page Flow editor is open:

  • Press Ctrl-Tab to change the focus to the Page Flow editor.
See the page view options

With the focus on the Page Flow Editor:

  • Press Z or z. The selected page view displays all navigation components on the page.
  • Press U or u. The selected page view hides all navigation components on the page.
  • Press Tab. The focus changes to the next page.

Note that the selected page has an orange border.
Switch between page views With the focus on a page in the Page Flow Editor:
  • Press Tab. The focus changes to the next page.
Delete a page connector

With the focus on the Page Flow editor:

  1. Press Tab to change the focus to the page that contains the connectors you want to delete.
  2. Press Delete to delete the selected page's connectors.
    If you want to delete the page from the project, click Yes in the Delete dialog.
Open selected page in the Design window

With the focus on the Page Flow editor:

  1. Press Tab to change the focus to the page that you want to open.
  2. Press Enter. The page opens in the Design window.
Create a navigation link

With the focus on the Page Flow editor:

  1. Press Tab to change the focus to the page that you want to use as the start.
  2. Press S. The selected page outline color changes to red.
  3. Press Tab to select the page to which you want to link.
  4. Press E to end the link. The linked pages outline colors change to orange.
  5. Edit the link name.
  6. Press Enter. The start page outline color returns to black.
Create a navigation link for a component

With the focus on the Page Flow editor:

  1. Press Tab to change the focus to the page that you want to use as the start.
  2. Press Z. The selected page view becomes larger.
  3. Press the up arrow or down arrow keys to select the button from which you want to link.
  4. Press S to mark the page and button as the start of the link.
  5. Press Tab to change the focus to the page to which you want to link the button.
  6. Press E to end the link. The linked pages outline colors change to orange.
Move a page

With the focus on the Page Flow editor:

  1. Press Ctrl-D to change to the draggable mode.
  2. Press Tab to select the page you want to move.
  3. Press the up arrow, down arrow, right arrow, or left arrow keys to move the selected page.
Set the layout mode to fixed

With the focus on the Page Flow editor:

  • Press Ctrl-D to change to the fixed mode.
Change the appearance of the lines

With the focus on the Page Flow editor:

  • Press Ctrl-S to change to straight lines.
  • Press Ctrl-W to change to wired lines.
Close the Page Flow editor
  • Press Ctrl-F4
