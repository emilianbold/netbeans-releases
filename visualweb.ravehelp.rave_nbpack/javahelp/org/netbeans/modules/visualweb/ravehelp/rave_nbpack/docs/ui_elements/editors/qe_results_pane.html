About the Results Pane

See Also

The results pane, the bottom pane in the Visual SQL Editor, displays the results returned when you test a query. To test a query, right-click in the SQL pane or diagram pane and choose Run Query. Then check the results pane to make sure that the query produced the results you intended.

  • If the SQL query contains a syntax error, the results pane displays nothing.
