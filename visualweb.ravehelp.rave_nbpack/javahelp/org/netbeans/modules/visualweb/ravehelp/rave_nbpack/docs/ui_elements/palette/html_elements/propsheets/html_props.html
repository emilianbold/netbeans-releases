html Component Properties Window

See Also

You see the Properties window for an html component when you select the component in the Navigator window. This component has the following properties.

General

  • id. Type: String
    The name of the html component in the page bean and the JSP file.

Advanced

  • lang. Type: String
    Code specifying the base language of the page. Equivalent to the lang attribute of the <html> tag. You must directly enter a value for the language code. Some examples:
    • ar - Traditional Arabic
    • de - German
    • en - English
    • en - English
    • es - Spanish
    • fr - French
    • ja - Japanese
    • en-US - United States English
    • ja-JP - Japanese
    • zh-guoyu - Mandarin or Standard Chinese
    • zh-Hans - Simplified Chinese script
    • zh-HanT - Traditional Chinese script

    For a longer list of two-character codes, see

  • rendered. Type: boolean
    If selected, the html component is to be rendered and processed on any subsequent page submission. If not selected, the <html> tag does not appear in the page at all.
  • xmlns. Type: String
    The XML namespace for the page, required because the page uses XHTML. Equivalent to the xmlns attribute of the <html> tag. The default value is http://www.w3.org/1999/xhtml. For more information on XML namespaces, see the following World Wide Web Consortium (W3C) web page:
