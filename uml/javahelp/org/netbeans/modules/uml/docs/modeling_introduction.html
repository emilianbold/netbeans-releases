About UML Modeling

The IDE's UML Modeling feature gives your software development team immediate visibility into your source code. The feature adds a set of powerful visual methods and aids for manipulating code, all directly within your existing development environment.

When UML Modeling is used to create blueprints of your software applications, each developer has a clearer road map and a shorter learning curve to implementation. Developers save time by navigating and manipulating code directly from the UML diagram. The application's code and its UML models are always in sync.
