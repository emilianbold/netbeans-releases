Configuration: UML Modeling

You can configure many UML Modeling preferences to suit your workflow and goals. The IDE offers a number of configuration settings from different locations, as shown in the following table:

Configuration Option Description
Setting UML Modeling Preferences You can customize UML Modeling to meet your specific needs. Use the IDE's Options window to set your UML preferences for global defaults and to edit default values. See UML Categories in the Options Window.
Specifying Layout Properties Opens the multi-tabbed Layout property sheet, which lets you specify the layout of your system diagrams in their respective Diagram windows.
Specifying Display and Drawing Preferences Opens the Display and Drawing Preferences dialog box, which lets you specify how the IDE handles zooming, perspective, resolution, and other universal display issues.
