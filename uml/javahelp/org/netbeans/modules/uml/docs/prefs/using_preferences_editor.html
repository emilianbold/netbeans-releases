Setting UML Modeling Preferences

See Also

You use the UML panel in the Options window to make changes to the default preferences that control the look and behavior of the UML Modeling features of the IDE.

The Options window displays the different categories of preferences that can be added, edited, or deleted. Select the category to display the its defaults in tabbed form.

To change UML Modeling preferences:

  1. Choose Tools > Options.
  2. In the Options window, click the UML category.
    The panel displays the following tabs where you can set the UML Modeling options.
    • General. This tab enables you to customize global settings for handling UML projects and diagrams.
    • Templates. - This tab enables you to customize the global settings to use when using domain templates
    • Java Code Generation. This tab enables you to customize the settings to use when the Java code is generated for your existing UML model
    • Dialogs. This tab enables you to customize whether a dialog appears for certain actions you take during UML Modeling.
  3. To change any of the UML preferences, select the appropriate tab, and then make your chages in the tab's pane.
  4. Click OK.
    Your changes are saved and the Options window closes.
