Allowing the Creation of Multiple Participants

If you want to be able to create more than one element with the same properties when applying your pattern, you must allow the creation of multiple participants.

To allow the creation of multiple participants:

  1. Select the role that can have multiple participants.
  2. In the Properties window, expand the Multiplicity row.
  3. Right-click in the Ranges row and select Create.
  4. Add a lower and upper bound for the range.
    • To allow the creation of two elements for the role, type 0 for the lower bound and 2 for the upper bound. To allow the creation of three elements, type 0 for the lower bound and 3 for the upper bound.
  5. To apply changes, deselect the role on the diagram.
  6. Save your project.
