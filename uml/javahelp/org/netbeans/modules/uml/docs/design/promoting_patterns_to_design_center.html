Promoting Patterns to the UML Design Center

If you want to apply a design pattern to multiple projects, the pattern must be in the design pattern catalog located in the open UML Design Center window.

To promote a pattern from a project in the Projects window to the UML Design Center:

  1. Open the user-defined project to which you want to add the pattern. For details, see Creating User-Defined Projects.
  2. In the Projects window, find the pattern that you want to promote.
  3. Right-click the pattern and choose Promote Design Pattern.
    The Promote Pattern to Design Center dialog box appears.
  4. From the Project list, select the project in the UML Design Center in which the pattern will be placed.
  5. If desired, select a package from the Namespace drop-down list.
  6. To remove the pattern from the project in the Projects window, select the Remove From Current Project checkbox.
    The pattern is removed from the Projects window and placed in the UML Design Center. If you don't select this option, the pattern is copied to the UML Design Center, but it also remains in the Projects window.
