Creating a Diagram for a User-Defined Pattern

The first step in building your own pattern is to create a diagram. Whether you build your pattern so that it appears in the Projects window or in the UML Design Center window, the process is the same.

  1. Right-click the project node and choose Add > Diagram.
    The Create New Diagram wizard is displayed.
  2. Select a diagram type.
    • Patterns can be created on both Class and Component diagrams.
  3. Enter a name for the diagram and click OK.
    The diagram is displayed in either the Projects window or the UML Design Center.
  4. To save your project along with the new diagram, right-click the project node and select Save.

Now you are ready to build the pattern.
