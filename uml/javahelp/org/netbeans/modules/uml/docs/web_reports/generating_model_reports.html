Generating Model Reports

See Also

UML Modeling offers a reporting feature that lets you generate a report detailing key information about your UML projects. The read-only report is produced in Javadoc format. It contains information about the packages, classes, interfaces, and other elements of your UML project.

To generate a UML Model report:

  1. Right-click the UML project from the Projects window.
  2. Select Generate Model Report from the pop-up menu.
    The report is generated in the background. A progress bar in the lower right corner of the IDE window shows that the process is running. The Output window opens a tab named UML Report Log and it displays the status of the report generation.
    Once generated, the report is displayed in a web browser.
  3. If necessary, cancel the report generation by clicking the progress bar.
