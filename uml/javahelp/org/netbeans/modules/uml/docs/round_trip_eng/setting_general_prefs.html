Setting General Code Engineering Preferences

See Also

Code engineering preferences can be set for general behavior during model element creation, deletion or manipulation.

To set general preferences:

  1. Choose Tools > Options.
  2. In the Options window, click the UML category.
  3. Click the Java Code Generation tab and define your preferences in the pane.
  4. Click OK to save your settings.
