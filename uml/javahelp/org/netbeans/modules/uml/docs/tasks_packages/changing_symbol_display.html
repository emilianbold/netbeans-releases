Changing Package Element Display

See Also

The IDE offers a modification feature in any Diagram window that lets you customize the display of a UML Modeling package name on the package element.

To change the display of package elements in a UML Diagram window:

  • In a Diagram window, right-click a UML Modeling package element. Under Show, do one of the following:
    • To display the package name on the element's tab, choose Name in Tab.
    • To display the package name in the center of the element, choose Name in Center.
    The IDE changes the display of the target package accordingly.
