Modifying Diagram Layouts

See Also

You can rearrange the existing layout of any UML diagram by applying one of several preset styles to the diagram. Each style adheres to a different layout algorithm.

You can use the buttons on the Layout toolbar to change the layout of a diagram. The IDE lets you configure most of the layout styles from the Layout Properties Sheet.

To modify the layout of a diagram:

  1. Click the target Diagram window.
  2. In the target Diagram window, right-click, and, under Layout, choose the target layout style.
    The IDE applies the layout style to all components in the target UML window.
