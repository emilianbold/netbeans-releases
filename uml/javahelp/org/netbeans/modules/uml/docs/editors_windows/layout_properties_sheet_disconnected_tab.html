Layout Properties Sheet: Disconnected Tab

See Also

The Disconnected tab of the Layout property sheet lets you specify options for all diagram elements that are not connected to other elements by a link. The table below describes the Disconnected tab options.

Section Option Description
Connected Components Detect Components Select the checkbox to have disconnected components (diagram elements) detected.
Constant Spacing Spacing between row and column components (diagram elements). Constant spacing affects the spacing around all elements in concrete units.
Proportional Spacing Spacing between row and column elements. Proportional spacing affects the spacing around all elements, proportional to each element's size.
Disconnected Nodes Detect Disconnected Nodes Select the checkbox to have disconnected nodes detected.
Constant Spacing Spacing between row and column elements. Constant spacing affects the spacing around all elements in concrete units.
Proportional Spacing Spacing between row and column elements. Proportional spacing affects the spacing around all elements, proportional to each element's size.
Aspect Ratio Automatic Whether to automatically maintain the ratio of width to length in a diagram's symbols, or to allow customization.
