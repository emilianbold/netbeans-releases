Specifying Display and Drawing Preferences

See Also

The IDE offers a Display and Drawing Preferences dialog box, which lets you specify how the IDE handles zooming, perspective, resolution, and similar display issues in a single Diagram window.

To specify display and drawing preferences:

  1. On the background of the target Diagram window, right-click, and then choose Properties.
    The IDE opens the Display and Drawing Preferences dialog box.
  2. Define your preferences and click OK or Apply to save your changes.
