Stacking Diagram Elements

See Also

The IDE lets you sort through stacked components. Use the stacking features available on the Diagram toolbar as follows:

  1. In a Diagram window, click a component.
  2. On the Diagram toolbar, click one of the following buttons:
    Button Description
    To send the component back behind one component in the stack
    To send the component to the back of the stack
    To bring the component forward one component in the stack
    To bring the component to the front of the stack
    The IDE moves the component accordingly.
  • You can also access stacking functionality from the Diagram menu, under Layout.
