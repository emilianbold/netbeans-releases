Filtering Element Types From the Projects Window

See Also

You can temporarily hide element nodes in the Projects window. For example, if your expanded project tree contains many Actors that you do not need to see at the moment, you can save space by filtering out Actor nodes. Or, if you want to flatten the appearance of your project tree, you can filter out Package nodes. You can select more than one element type to filter out.

  • After you apply a filter to the Projects window, all element types still appear in the Diagram editor and other windows.

To filter out element types:

  1. In the Projects window, open a project, right-click the Model node, and choose Filter.
    The Filter dialog box appears.
  2. Expand the Model Elements node and select the element types that you want to filter out.
    A checkmark indicates that the element type will not be filtered out. When you click an element type, the checkmark disappears. When you click the element type again, the checkmark appears.
  3. Click OK.
    The IDE applies your filter to the Projects window. The Projects window now displays all element types, except the ones you selected to be filtered out.
