Display and Drawing Preferences Dialog Box

See Also

The Display and Drawing Preferences dialog box lets you specify several options. This topic describes these options and the command buttons on the dialog box.

Options

  • Graph Window
    • Auto Hide Scroll Bars. This checkbox lets you specify whether to hide or reveal Diagram window scroll bars when entire diagrams are shown. By automatically hiding unnecessary scroll bars, you gain more image space in Diagram windows.
    • Show Tool Tips. This checkbox lets you show or hide tool tips.
    • Min Zoom Percent. This box lets you specify the minimum scale beyond which the IDE will no longer zoom in.
    • Max Zoom Percent. This box lets you specify the maximum scale beyond which the IDE will no longer zoom in.
    • Undo Stack Size. Enter the number of levels to undo.
  • Compress Meta Edges. This specifies whether multi edges are merged into a single meta-edge on collapse operations.
  • Geometric Adjustment. Specifies whether to adjust the spacing of objects within a drawing after expand, collapse, fold, unfold, hide and unhide commands complete their operations.
  • Tools
    • Partial Selection. This specifies whether or not an object that is partially inside the marquee selection rectangle gets selected.
    • Opaque Movement. This specifies whether the whole object or only the outline is drawn on dragging.
    • Edge Hit Tolerance. This specifies the assumed width of edges for the purpose of hit-testing.
    • Reconnect Edge Sensitivity. This specifies the threshold which determines when the reconnect edge tool should be activated.
    • Interactive Zoom Sensitivity. This specifies the sensitivity of the mouse when zooming interactively.
    • Panning Sensitivity. This specifies the sensitivity of the mouse when panning.
    • Edge Navigation Preferences
      • Trip Duration. This specifies the duration of the trip from one end node to the next in link navigation.
      • Blink Duration. This specifies the duration of blinking of edges as they are being cycled through.
      • Frame Count. This specifies the number of frames (or steps) used to scroll from one end of the edge to the other.

Command Buttons

  • OK. Accepts your specifications and closes the dialog box.
  • Cancel. Closes the dialog box without applying any changes made in the Display and Drawing Preferences dialog box.
  • Apply. Applies your specifications without closing the dialog box.
  • Reset. Resets all values in the dialog box to the values that were displayed when you opened the dialog box.
  • Defaults. Restores all values in the dialog box to their original settings.
