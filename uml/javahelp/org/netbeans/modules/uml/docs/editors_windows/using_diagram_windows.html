Using the Diagram Editor

See Also

The Diagram editor is where, if you are doing forward engineering, you populate each diagram in a UML Model. Or, if you have reverse-engineered a diagram from existing code, the Diagram editor is where you can display and continue to work on a diagram.

To populate a diagram with new modeling elements, you drag element icons from the Modeling Palette into the Diagram editor. Or, to reuse an element from another diagram, you drag the element's node from the Projects window into the Diagram editor.

In the Diagram editor, you can also use the following navigation features:

  • Zoom into and out of diagrams
  • Stack diagram elements
  • Modify diagram layouts
  • Specify layout properties
  • Specify display and drawing preferences
  • Navigate within diagrams
  • The IDE offers a tabbed interface at the top of the Diagram editor. Select a diagram's tab to bring its window to the top.
