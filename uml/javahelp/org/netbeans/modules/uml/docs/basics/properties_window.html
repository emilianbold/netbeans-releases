Properties Window

See Also

The Properties window displays details about the properties of any modeling element you select in the Diagram editor or the Projects window.

To edit a property in the Properties window, double-click a row, use the drop-down list, or edit the field directly.

UML properties are described in the read-only description space at the bottom of the Properties window.
