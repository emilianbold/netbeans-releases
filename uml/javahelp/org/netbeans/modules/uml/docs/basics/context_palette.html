Context Palette

See Also

Many functions in the Diagram Window can be accomplished by right-clicking on an edge. This causes a Context Palette to appear, which has different functions, dependent on the type of diagram. Clicking on one of the appropriate icons in a Context Palette causes an action to occur.

The following topics describe the elements available in the Context Palette for each type of diagram:

  • Activity Diagram Elements
  • Class Diagram Elements
  • Sequence Diagram Elements
  • State Diagram Elements
  • Use Case Diagram Elements

Notes on Context Palette Use

When you use the keyboard to perform Context Palette actions, ensure that you are using the Context Palette correctly.

For example, to create a message within a Sequence Diagram, using the keyboard:

  1. Create a new Sequence Diagram.
  2. Within the Diagram, create a newline as an originated lifeline for a Create Message.
  3. Create a second lifeline in the Diagram; this lifeline can not have other message
  4. Select the 1st and 2nd lifelines
  5. Select the Up/Down arrow keys to move the icon selection to the Create Message icon on the context palette.
  6. Press Ctrl+Enter to perform the "Create Message".
