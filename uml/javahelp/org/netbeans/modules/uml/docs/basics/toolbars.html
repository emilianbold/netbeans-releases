Diagram Toolbar

See Also

The area at the top of the Diagram editor contains a toolbar whose buttons let you manipulate diagrams and navigate among their elements.

If an element of the Diagram toolbar is hidden, expand the IDE window, resize the Modeling Palette, or slide the Modeling Palette over to the edge.

The Diagram toolbar contains the following sections:

  • Export Button
  • Standard Buttons
  • Diagram Properties Buttons
  • Stacking Buttons
  • Zoom Buttons
  • Layout Buttons

Export Button

Component Toolbar Icon Description

Export as Image (Ctrl-Shift-X)

Export the diagram as a JPEG image, a Portable Network Graphics (PNG) image, or a Scalable Vector Graphic (SVG) image.

Standard Buttons

The following table describes standard navigation features available from the toolbar:

Component Toolbar Icon Description
Select (Ctrl-Shift-S) Click this button to return the cursor to its default mode.
Pan (Ctrl-Alt-Shift-P) Lets you move the entire diagram around in the Diagram editor.
Navigate Link (Ctrl-Alt-Shift-L) Lets you click any line in any diagram to immediately navigate to one of its components or lifelines.
Overview Window (F8) Lets you display an entire diagram in an inset window, scaling the view as necessary.

Diagram Properties Buttons

The following table describes the diagram properties that you can manipulate from the toolbar:

Component Toolbar Icon Description
Diagram Synchronization (Ctrl-Alt-Shift-R) Use this button when the information displayed in a diagram is out of synch with the metadata in the .edt file.
Relationship Discovery (Ctrl-Shift-D) The IDE discovers and displays project relationships between elements, if those relationships were not displayed before.

Stacking Buttons

The following table describes the stacking features available from the toolbar:

Component Toolbar Icon Description
Move Forward (Ctrl-Alt-Shift-U) Bring the element forward one element in the stack
Move to Front (Ctrl-Alt-Shift-F) Bring the element to the front of the stack
Move Backward (Ctrl-Alt-Shift-B) Send the element back behind one element in the stack
Move to Back (Ctrl-Alt-Shift-D) Send the element to the end of the stack

Zoom Buttons

The following table describes the fit and zoom navigation features available from the toolbar:

Component Toolbar Icon Description
Fit to Window (Ctrl-Shift-F) Automatically brings an entire diagram into full view in the Diagram editor, offering a good overview of the components that make up the diagram.
Custom Zoom Lets you custom-zoom into and out of any diagram by specifying a zoom percentage.
Zoom with Marquee (Ctrl-Alt-Shift-Z) Lets you zoom into a diagram by specifying an area for zooming. You can drag a box or marquee over the diagram.
Zoom Interactively (Ctrl-Alt-Shift-I) Lets you zoom into and out of a diagram by dragging your pointer along the diagram's y-axis.
Zoom In (Alt+) Lets you incrementally zoom in and enlarge your diagram gradually.
Zoom Out (Alt-) Lets you incrementally zoom out and reduce the size of your diagram gradually.

Layout Buttons

The following table describes the layout features available from the toolbar:

Component Toolbar Icon Description
Hierarchical Layout (Ctrl-Shift-K) Rearranges your diagram into a hierarchy pattern, with most elements falling below a lead element.
Orthogonal Layout (Ctrl-Shift-B) Rearranges your diagram into a square-lined, roughly rectangular pattern.
Symmetric Layout (Ctrl-Shift-Y) Rearranges your diagram into a symmetrical pattern centered around a single element.
Incremental Layout (Ctrl-Shift-I) Rearranges your diagram without disturbing styles that you already applied to pre-existing diagram elements.
