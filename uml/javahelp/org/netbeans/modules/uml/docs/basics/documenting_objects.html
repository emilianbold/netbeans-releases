Documenting Project Elements

See Also

You can use the Documentation window for documenting projects, diagrams, and their components.

The Documentation window:

  • Is accessible as a persistent IDE window, letting you document on the fly whenever you select an item in the Projects window or Diagram editor.
  • Lets you document any project, diagram, or element that you select in the Projects window.
  • Offers no Tag support.

To use the Documentation window:

  1. Select the project or any node under it in the Projects window. Or, select any element in the Diagram editor.
    The element's icon and name appear in the Documentation window.
  2. In the Documentation window, type or edit the documentation you want.
  3. Customize your documentation using the features of the Documentation window.
    The Documentation window redisplays your documentation each time you select the element in the Diagram editor or the element's node in the Projects window.
