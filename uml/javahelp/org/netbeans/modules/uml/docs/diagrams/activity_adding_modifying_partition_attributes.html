Adding and Modifying Partition Attributes

See Also

The IDE lets you add and modify two types of Partition attributes: dimension and external Boolean.

Dimension Boolean

A dimension Boolean attribute indicates whether the Partition groups other Partitions along a dimension.

To add a dimension Boolean attribute:

  1. In the Diagram editor, select a Partition.
  2. In the Properties window, select the Dimension list, and then choose True or False.

External Boolean

An external Boolean attribute indicates whether the Partition represents an entity to which the participating structure does not apply.

To add an external Boolean attribute:

  1. In the Diagram editor, select a Partition.
  2. In the Properties window, choose the External list, and then choose True or False.
