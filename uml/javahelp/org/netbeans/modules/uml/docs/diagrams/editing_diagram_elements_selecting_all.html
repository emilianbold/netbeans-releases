Selecting All Elements

You can select all components in a UML Diagram window. When you select a single component (by clicking the component), the IDE displays handles around the selected graphic. Likewise, when you select all components, the IDE displays handles around every component in the window, letting you cut, copy, delete, or otherwise work with all components at once.

To select all elements in a Diagram window:

  • Right-click the target component, and choose Select All.
    The IDE selects all components in the window, displaying handles around each component.
