Using the Compartment Customize Dialog Box

UML Modeling offers the Compartment Customize dialog box that lets you customize the look of the compartments in any individual Class component. You can use this dialog box to show and hide all, or any selection of, attributes and operations displayed in a target Class component's compartments.

To use the Compartment Customize dialog box:

  1. In the target Diagram window, right-click the target Class component, and then, under Compartment, choose Customize.
    The Compartment Customize dialog box opens.
  2. To hide any attribute or object displayed in the target Class component, in the Available Compartments box, clear the checkbox of the individual attribute or operation.
    • You can also hide all displayed attributes or operations by clearing the Attributes or Operations node.
  3. To show any hidden attribute or object, select the appropriate checkbox.
    • You can also show all available attributes or operations by selecting the Attributes or Operations node.
  4. Click OK.
    The IDE accepts your customizations, and refreshes the display of the target Class component to reflect the changes.
