Reordering and Moving Attributes

UML Modeling offers a drag feature in the project tree that lets you reorder the list of attributes appearing in any class component, or move any attribute to a different class.

  • When moving attributes from one class to another, UML Modeling also moves any corresponding operations.

To reorder attributes:

  1. In the Attributes section of the target class on the Project tree, choose the target attribute.
  2. Drag the attribute anywhere in the Attributes section of the source class Project tree.
    UML Modeling reorders the Attributes list in the Project tree.

To move attributes:

  1. In the Attributes section of the source class component, choose the target attribute.
  2. Drag the attribute in the Attributes section of the target class component on the Project tree.
    UML Modeling removes the attribute from the source class on the Project tree and displays it in the target class on the Project tree.,.
