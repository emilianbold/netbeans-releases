Working With Association Names

When you name a Class diagram's association, the name appears in the Projects window, the Properties window, and the association's Class diagram. You can show or hide an association name in the Diagram editor.

To show or hide an association name:

  • In the Diagram editor, right-click the association, select Labels, and then choose Link Name.
    The name is displayed or hidden, as applicable.

To name or rename an association:

  1. In the Diagram editor, double-click the association name.
    The name becomes editable.
  2. Type the association name, and then click the diagram background.
    The new name is displayed in the Diagram editor, the Projects window, and the Properties window.
  • You can also insert or change an association name using the Properties window. Select the association in the Diagram editor. The association's properties appear in the Properties window. Type a new name into the Name field, or double-click and edit the Name field.
