Hiding and Showing Attributes in a Class Diagram

The IDE's UML Modeling feature lets you hide and show the attributes section of any class component in any Class diagram.

To hide or show attributes:

  1. On the Class element, click the line that marks the upper border of the Operations sections.
    The pointer display changes, indicating that the line is ready to move.
  2. To hide the Attributes section, move the line up until the section is hidden.
  3. To show the Attributes section again, double-click the line that separates the Attributes and Operations sections.
  • You can also show and hide the Attributes section by Using the Compartment Customize dialog box.
