Changing Diagram Element Size

The IDE lets you change the size of any component in a UML Diagram window. Once changed, the IDE lets you change the size of the component so that all information is displayed without adding unnecessary space.

To change the size of a component:

  1. In the component's Diagram window, select the component.
  2. Drag the component handles to change the size.
    The IDE displays the component at its specified size and proportions.

To shrink a resized object so that its data is viewable but extra space is removed:

  • Right-click the target component, and then right-click on the element to choose Resize Element(s) to Contents.
    The IDE displays the component at a size that makes all data easily viewable.
