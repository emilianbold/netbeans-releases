Making an Interruptible Activity Group

See Also

The procedure in this topic demonstrates how you can work with Interruptible Activity Groups in the UML Modeling feature. The following procedure uses an example with a partition named Banking and a subpartition named ATM.

To make an Interruptible Activity Group:

  1. Create a partition and name it Banking.
  2. Add a subpartition to the right of the partition and name it ATM.
  3. In the Modeling Palette, click the Activity Group icon .
  4. To nest the Activity Group inside the ATM subpartition, do as follows:
    • In the Diagram editor, click the ATM subpartition.
      The new Activity Group appears inside the subpartition.
  5. Name the Activity Group by doing one of the following:
    • In the Diagram window, double-click the new, unnamed Activity Group, and type the name Dialog.
    • In the Properties window, in the Name box, type the name Dialog.
  6. In the Diagram editor, select the Activity Group Dialog.
  7. In the Properties window, choose the GroupKind list, and then choose Interruptible.
