Locking Diagram Elements

You can disable in-place editing of element attributes and operations in the Diagram editor by using an editing lock on the diagram element.

When you double-click a locked diagram element that exists in more than one diagram, the Element Navigation dialog box displays.

  • Using an editing lock on a diagram element does not prevent you from selecting the element and typing a new name for that element.

To set or remove an editing lock on a diagram element:

  1. In the Diagram editor, select a diagram element and right-click.
  2. On the pop-up menu, select Edit > Lock Edit.
    If the diagram element was unlocked, the IDE sets the editing lock.
    If the diagram element was locked, the IDE removes the lock.
