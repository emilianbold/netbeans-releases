Creating an Empty Diagram

See Also

The IDE lets you create new, empty UML diagrams in any open project in the Projects window. You decide on the project where you want the diagram stored, and you use the New wizard. Then you populate your diagram with UML Modeling elements.

To create a new, empty diagram:

  1. In the Projects window, expand the project node.
  2. Right-click the Model node and choose Add > Diagram.
    The New - Create New Diagram wizard opens.
  3. In the Diagram Type pane, select a diagram type.
  4. In the Diagram Name field, type a name for the new diagram.
  5. In the Namespace box, select the name of the project that you want to store the diagram.
    • By default, a diagram is created under the node currently selected in the Projects window.
  6. Click OK.
    The new diagram is added under the project node in the Projects window, and a new empty diagram window opens in the Diagram editor.
