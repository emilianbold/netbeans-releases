Selecting and Modifying an Activity Group Kind

See Also

An Iteration Activity Group has two Kinds: a test to continue the loop is done at the beginning (testAtBegin) or end of the loop (testAtEnd).

To select and modify an Activity Group Kind:

  1. In the Diagram editor, select the Activity Group element.
    The Properties window shows the Activity Group details.
  2. From the Kind list, select a Kind.
