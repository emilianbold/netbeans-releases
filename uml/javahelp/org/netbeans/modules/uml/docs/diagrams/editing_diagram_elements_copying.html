Copying Elements

You can copy any UML modeling element that is displayed in the Diagram editor. When you copy an element, the IDE removes it from the window and places it on a clipboard. You can then paste the copied element into the same diagram or a different diagram that is open in the Diagram editor, or into selected third-party applications. The copied element remains on the clipboard until you cut or copy another selection.

To copy and paste an element:

  1. Right-click the UML modeling element and choose Copy.
    The IDE copies the element to a clipboard for pasting.
  2. Right-click in the destination file and choose Edit Paste.
    A Paste symbol appears.
  3. Position the Paste symbol and click once.
    The element you pasted appears in the target diagram or application.
