Assigning a User-Defined Multiplicity Value to an Association

The IDE lets you assign user-defined multiplicity values to an association from its Diagram window. You can assign these values in the Diagram window only when the association multiplicity value is shown.

To assign a new user-defined multiplicity value:

  1. In an association Diagram window, double-click the association's multiplicity value.
    The IDE makes the value editable.
  2. Type the association new multiplicity value, and then click the diagram background.
    The IDE accepts the value and displays it at the specified end of the association, as well as under the Ends node of the Properties window.

    The IDE also lets you specify an association's multiplicity value from the Properties window. Select the association end to display its information in the Properties editor, and then edit it.
