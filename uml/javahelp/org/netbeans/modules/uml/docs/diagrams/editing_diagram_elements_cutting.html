Cutting Elements

You can cut any element that is displayed in the Diagram editor. When you cut an element, the IDE removes the element from the editor window and places it on a clipboard. You can then paste the element into the same diagram, into a different diagram, or into selected third-party applications like Microsoft Word. The cut element remains on the clipboard until you cut or copy another selection.

To cut an element:

  • Right-click the element and choose Edit > Cut.
    The IDE removes the element from the Diagram editor and places it on a clipboard for possible pasting.
