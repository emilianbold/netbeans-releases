Associating Elements

See Also

The UML Modeling feature includes the Associate dialog box, which lets you associate any modeling element with any other element. You can also associate an element with a diagram, and a diagram with another diagram. The Associate dialog box is like the Find dialog box.

To associate elements:

  1. Select an element in one of the following places :
    • In the Projects window
    • You must select an element to open the Associate dialog box.
  2. Right-click the element and choose Associate With.
    The Associate dialog box opens.
  3. Specify the Associate With parameters, and then choose Find.
    The IDE searches the current project and returns results in the grid.
  4. In the grid, select the element(s) to associate, and then choose Associate or Associate All.
    The IDE associates the elements, and creates a Reference in the Projects window and the Properties window.
