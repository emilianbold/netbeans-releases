Working With Diagrams and Elements

See Also

The IDE offers several methods of working with the elements that make up your diagrams.

  • UML Modeling's inter-diagram synchronization is bi-directional. If you change a method name or a return type on a Class diagram, the message is updated automatically on corresponding Sequence diagrams. As you add messages between elements, the appropriate classes on the Class diagram are updated automatically with methods. This makes implementation easier, and takes care of tedious work that otherwise would have to be done by hand.
