Using the Redefine Dialog Box

In the IDE's UML Modeling feature, you can perform redefinitions using the generation link.

Once you draw the link, a Select Methods to Redefine dialog box opens that lets you select attributes and operations to be inherited from the parent class. Do as follows:

  1. Click the box to the left of the selected class component to select all the methods.
    All methods in the list are selected.
  2. Click OK.
    The dialog box closes, and all selected methods are added to the selected class.
