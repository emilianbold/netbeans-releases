Naming Component Elements

See Also

You can name UML Component elements on a Component diagram and in the Properties window.

To name a Component element on a Component diagram:

  • In the Diagram editor, click the target Component element to select it and type the name.

To name a Component element in the Properties window:

  1. In the Diagram editor, select the target Component element.
  2. In the Properties window, click the Name row, and type the new name.
