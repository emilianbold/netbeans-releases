Renaming Diagram Elements

The IDE lets you rename diagram elements from the Projects window, the Diagram editor, or the Properties window.

To rename a diagram element from the Projects window:

  1. Right-click the element node and choose Rename.
    The Rename dialog box appears.
  2. Type a new name and press enter. The IDE displays the new name on the node in the Projects window, and on the element in the open diagram.

To rename a diagram element from the Diagram editor:

  1. Select the element and click twice.
    The IDE makes the element name editable.
  2. Type the new name, and then do one of the following:
    • Click outside the element.
    • Press Enter.
    The IDE displays the new name on the element in the Diagram editor and on the element's node in the Projects window.

To rename a diagram element from the Properties window:

  1. In the Projects window or the Diagram editor, select an element.
    The IDE displays the element's details in the Properties window.
  2. In the Properties window, select the Name field.
    The IDE makes the name editable.
  3. Type the new name and then do one of the following:
    • Click outside the Properties window.
    • Press Enter.
    The IDE displays the new name in all three places.
