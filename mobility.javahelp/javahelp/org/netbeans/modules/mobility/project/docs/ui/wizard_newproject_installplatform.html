Install Platform Page

You open the New Mobile Project wizard by choosing File > New Project and selecting a project from the Mobile category.

The Install Platform page is called if there is no Java ME MIDP platform-compatible SDK or , such as the Sun Java Wireless Toolkit, installed in the IDE.

To add an SDK or emulator platform:

  1. Click the Install SDK/Platform/Emulator button.
  2. Use the Add Platform wizard to install an emulator into the IDE.

After an emulator platform is installed, you are returned to this page to continue adding a new project. Click Next to continue.
