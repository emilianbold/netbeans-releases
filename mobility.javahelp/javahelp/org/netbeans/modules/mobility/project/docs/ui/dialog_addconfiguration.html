Add or Duplicate Project Configuration Dialog

See Also

Use the Add or Duplicate Configuration dialog to add new to the active selected project. You open the Add Configuration dialog by choosing Add Configuration from the Project Configuration menu of the Project Properties, or clicking the Add button in the Project Configuration Manager.

You open the Duplicate Configuration dialog by clicking the Duplicate button in the Project Configuration Manager.

In this dialog, enter a name for a new configuration, then click OK. The configuration name is added to the project.

If you are adding a configuration, the new configuration will have the same properties as DefaultConfiguration. Optionally, you can choose a project template from the Use Configuration Template dropdown menu. The new configuration contains all the properties of the configuration template.

You can delete a project template by clicking the Delete button.

If you are duplicating a configuration, the new configuration will have the same properties as the configuration you are duplicating.

You can customize the properties for the new configuration by choosing property sheets from the menu tree in the right pane of the Properties dialog.
