New Ability

See Also

Use the New Ability dialog to create new abilities. The new ability can then be associated with the current project configuration.

You open the New Ability dialog by clicking the New Ability button in the Add Ability dialog.

To create a new ability:

  1. Enter a new name for the new ability. and click OK.
    The ability is added to the list of available abilities in the Add Abilties dialog. .
