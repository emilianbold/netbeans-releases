Java Specification Request (JSR) 172

JSR-172 (J2ME Web services APIS) is the standard API for accessing Web services from a Java ME platform-enabled mobile phone. It adds Java ME platform support for XML parsing and SOAP/XML-baased web services.

JSR-172 specifies two new optional packages:

  • an XML parser that is a subset of JAXP functionality
  • a Web services remote invocation API
