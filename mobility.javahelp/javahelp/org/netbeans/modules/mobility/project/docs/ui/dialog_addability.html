Use the Add Ability dialog to associate an ability with a project configuration or to create new abilities.

You open the Add Ability dialog by clicking the Add Ability button in the Abilities Property page.

To associate an ability with the current project configuration:

  • Choose an ability from the list and click OK.
    • The Abilities drop-down menu lists all the abilities that have been created in the IDE, even for projects that are closed.

To create a new ability:

  1. Click the Add Ability button.
  2. Choose an Ability name from the drop-down menu.
  3. Optionally, add a value for the ability. Then click OK.
    The ability is added to the list of abilities.
