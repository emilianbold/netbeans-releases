Import Mobile Project from Existing MIDP Sources Wizard:
Specify MIDP Sources page

See Also

You open the Import Mobile Project from Existing MIDP Sources wizard by File New Project. In Categories, select Mobility. In Projects, select Mobile Project from Existing MIDP Sources.

In the Specify MIDP Sources page, you set the following properties:

  • Sources Location. Specifies the root folder (the folder above the first package) that contains the Java source code for the project.
  • JAD/Manifest Location. Specifies the file that contains the Java Application Descriptor (JAD) for the project. JAD files are indicated by the extension .jad.

Click Next to continue to the Project Name and Location page.
