Project Properties

See Also

You open the Project Properties by choosing File > <active project name> properties. This opens the Project Properties dialog.

Use the Project Configuration drop-down menu to choose the active for your project. You can then customize the properties for this configuration by choosing property sheets from the menu tree in the left pane of the Properties dialog.

Choose Add Configuration from the drop-down menu to add a new configuration to the project.

Click the Manage Configurations button to open a dialog that displays the complete list of configurations created for the selected project. In the Project Configuration Manager dialog, you can add, duplicate, or remove configurations from the project, or create a project configuration template.
