About Over-The-Air (OTA) Provisioning

See Also

Over-the-Air (OTA) provisioning is the process of deploying an application from a website on to a mobile device. If you choose Execute Through OTA in the MIDP Project Running Property sheet, the IDE simulates the OTA process so you can test and demonstrate the full provisioning process of the MIDlet suites from the server to a device.

The MIDP specification includes the document, "Over The Air User Initiated Provisioning Recommended Specification" describes how MIDlet suites can be deployed over-the-air (OTA), and the functions that a device should provide to support such deployments. You can download this document from .
