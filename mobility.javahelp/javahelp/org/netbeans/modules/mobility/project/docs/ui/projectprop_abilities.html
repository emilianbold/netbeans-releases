MIDP Project Properties: Abilities Page

See Also

Use the Abilities property page to create new abilities and associate them with .

You open the Abilities property page by choosing File > <active project name> Properties. In the Project Properties dialog, select Abilities.

  • If you are setting the properties for a project configuration other than the default configuration (DefaultConfiguration), you must uncheck the Use Values from "DefaultConfiguration" checkbox at the top of the page to customize these options.

The Abilities window lists abilities that are associated with the currently active project configuration.

To associate an ability with the current project configuration:

  1. Click the Add Ability button.
    This opens the Add Ability dialog.
  2. Choose an ability from the list and click OK.
    • The Abilities drop-down menu lists all the abilities that have been created in the IDE, even for projects that are closed.
    • Optionally, add a value for the ability.

To remove an association with a project configuration:

  • Click the Remove Ability button.
