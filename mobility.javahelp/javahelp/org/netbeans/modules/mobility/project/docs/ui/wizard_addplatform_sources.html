The Add Java Platform wizard enables you to register Java platform and Java ME, MIDP platform-compatible SDKs or , such as the Sun Java Wireless Toolkit, in the NetBeans IDE. Once you register a platform emulator, you can use to set that platform for compilation, execution and debugging.

Use the Sources and javadoc page of this wizard to specify the classpaths of source files and javadoc used by a custom (non-UEI compliant) emulator platform.

After you have entered the information required by the Bootstrap Libraries page, detects the libraries and APIs used by the platform.

You can add additional source files and javadoc files by clicking the Browse button and navigating to the files you want to add.

Click Finish to register the emulator platform in the IDE.
