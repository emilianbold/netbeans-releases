New Embedded Application Wizard:
More Configurations Selection Page

See Also

You open the New Embedded Application wizard by choosing File > New Project and selecting Embedded Application from the Java ME category.

In the More Configurations Selection page, you can choose additional for the project. The configurations are then immediately available to you for testing and compiling your project.

The list of configurations is created from all the configuration templates that have been created in the IDE.

  • After you create the project, you can create additional project configurations with alternate platform settings.

Clicking the Finish button completes the wizard and creates a project you can view in the Projects window of the IDE.
