Visual Mobile Designer: Debugging View

See Also

The Activate Visual Designer Debugging Views checkbox activates the Document and Registry views in the Visual Mobile Designer.

  • Document view: When this view is enabled for a visual MIDlet it displays the structure of the design documents with all of the design components.

  • Registry view: Displays the registered component descriptors and component producers for the visual MIDlet when enabled.
