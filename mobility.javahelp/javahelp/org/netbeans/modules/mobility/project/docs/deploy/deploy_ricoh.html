Deploying CDC Projects to Ricoh Devices

See Also

You can deploy your CDC application to a Ricoh device using one of the two following methods. Before attempting to deploy the application, make sure the CDC project ran successfully on the target device emulator.

Using an SD card to deploy to a Ricoh device:

  1. Save the application to an SD card suitable for your Ricoh device.
  2. Put the SD card into the device's SD card slot.
  3. Follow the device's instructions to load and run the program.

Using HTTP POST to deploy to a Ricoh device:

  1. Connect to the Ricoh device's IP address on your local network.
  2. Log in to the device if necessary.
  3. Use HTTP POST to transfer the application from your computer to the device and run the program.
