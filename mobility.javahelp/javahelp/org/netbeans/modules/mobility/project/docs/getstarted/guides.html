Mobile Application Development Tutorials and Guides

Java ME application development is different from developing Java SE projects in many ways. The NetBeans web site has several resources for learning more about Java ME features and how to use them for mobile application projects.

Learning Trail


A series of thematically arranged tutorials that provide a foundation for developing mobile applications using NetBeans Java ME support.

Introductory Guides


Shows you how to create, build and run a simple "Hello World" MIDlet using the Visual Mobile Designer or project template. The guide also links to tutorials that show you how to create preprocessor blocks for multiple device configurations and how to generate mobile client/server applications using the Wireless Connection Wizard.

Shows you how to create, build and run a simple "Hello World" MIDlet for CDC devices and links to tutorials that show you how to make a simple GUI using Project Matisse, set up the CrEme VM to run Java Swing applications on Windows CE devices as well as other IDE features that support Java ME CDC development.
