Choosing Project Configurations

See Also

The Build Selected Project Configuration dialog allows you to choose which to build for your project. The IDE then builds a distribution JAR for each configuration you select.

  • To add configurations check the box next to the device you want to include in you project's Batch Build.
  • To remove configurations uncheck the box next to the device you want excluded from the build.
