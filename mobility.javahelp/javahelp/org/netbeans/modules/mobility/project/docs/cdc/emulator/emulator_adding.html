Adding Emulator Platforms

See Also

Adding an emulator platform to the IDE enables you to use that platform for compilation, execution and debugging within the IDE.

The Sun Java Toolkit for CDC 1.0 is integrated within the NetBeans Mobility Pack. However, if you want to program applications for other supported platforms, you need to download and install one or more of the following SDKs:

  • Ricoh Embedded Software Architecture Emulator
  • Sony Ericsson
    • UIQ 3 SDK
    • M600 Extension Package for the UIQ 3 SDK
    • P990 Extension Package for the UIQ 3 SDK
  • Nokia Series 80 Platform SDK for Symbian OS, for Java, Personal Profile
  • The SavaJe Operating System (OS)
