Viewing the General Mobility Project Properties

See Also

You can view the general properties of a Mobility project by choosing File > "active project name" properties. In the Project Properties dialog, select the General category.

The General project properties displays the project name and Ant project name (folder name) for the active project. It also displays a list of projects the active project depends on. These fields cannot be edited.

You can check or uncheck the Rebuild Required Projects check box. If this box is checked, the required projects will be rebuilt before the active project is built.
