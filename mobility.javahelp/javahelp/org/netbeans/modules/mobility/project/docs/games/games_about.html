About the Game Builder

See Also

The Game Builder provides a visual user interface to the Mobile Game API package defined in JSR-178 and enables you to create games for Java enabled mobile devices. In addition to the Game Builder Designer, a Sample Game is included in the IDE that can be used as a model for further development.

The following API's are featured in the :

  • Scenes
  • Tiled Layers
  • Sprites
