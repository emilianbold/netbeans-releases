Customizing a Debugger Window

See Also

To add or remove a column to a window:

  1. Click the  icon in the upper right corner of the window, or right-click in the window and choose List Options > Change Visible Columns.
  2. In the Change Visible Columns dialog box, click a checkbox to turn the display of information on or off.
  3. Click OK.

    The column of information is added or removed from the window.

To rearrange the columns in a window:

  • Drag the column header to the right or left and release the mouse button.

    The column is placed at the new location.

To set the sort order of a window:

  • Right-click anywhere in the window and choose List Options > Sort > sort-order.

    For all windows, you can sort the column in ascending or descending order. A triangle is displayed in the column header. The direction in which the triangle is pointing indicates whether the sort order is ascending or descending. Some windows provide additional sort orders.
