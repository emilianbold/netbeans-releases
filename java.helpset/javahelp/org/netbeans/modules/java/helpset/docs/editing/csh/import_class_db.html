Import Class Dialog Box

See Also

Fast import enables you to quickly add import statements for Java classes in your file.

To access the Import Class dialog box, move the insertion point to the name of a class in your Java source file and press Alt-Shift-I. The Import Class dialog box displays a list of classes that match the class at the insertion point.

You can select one of the following radio buttons:

  • Import Class. Generates the import statement for the class in the code.
  • Import Package. Generates the import statement for the package in the code.
  • Fully Qualified Name. Generates the fully qualified name in the code.
