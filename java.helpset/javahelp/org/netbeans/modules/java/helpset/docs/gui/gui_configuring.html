Configuring the GUI Builder

See Also

You can adjust how the IDE's GUI Builder displays form information, edit the layout grid size, and assign specific editors for certain property types, among other things.

  • If you want to edit settings for a particular form, select the form's root node in the Inspector window and click the Code button in the Properties window.

To configure GUI Builder settings:

  1. Choose Tools > Options from the main window.
  2. Click Miscellaneous in the left pane of the Options window.
  3. Click the GUI Builder node in the right pane to display the editable GUI Builder Settings.
  4. Select and edit the property you wish to change in the right pane of the Options window.

    The IDE applies the new settings to the GUI Builder.

  • If you cannot find the GUI Builder setting you want to edit, click the Advanced Options button to view a list of additional editable properties.
