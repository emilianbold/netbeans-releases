Connecting Composite Application Components

See Also

The Design View of the CASA Editor displays composite application components and their connections. You can perform the following operations on connections between components:
  • Delete a Connection
    Select a component and press the Delete key. You can also right-click a component, and select Delete.
  • Add a Connection
    From either a Consume or Provide endpoint of a component, select the endpoint and drag it to a Provide or Consume endpoint on another component. If the connection satisfies all validation rules, the connection is created.

    Note: A provider endpoint can have multiple connections from multiple consumers, but a consumer endpoint can have only a single connection with a single provider. For example, if a WSDL binding is already connected to a provider, you have to first delete this connection before you can connect it to another provider.
  • View or Modify the Properties of a Connection
    Select a component to view its properties in the Properties window. Use the Properties window to modify any of its properties.
