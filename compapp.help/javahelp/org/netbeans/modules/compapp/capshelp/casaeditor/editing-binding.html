Adding and Configuring WSDL Ports

See Also

Use the CASA Editor to add and configure the WSDL ports of a composite application.

To add and configure a WSDL port:

  1. From the Palette, select the icon for the WSDL Binding you want to add.
  2. Drag the icon for the binding to the WSDL Ports section of the Design Area.
    A representation of the binding appears in the WSDL Ports section.
    Note: You can also right-click in the WSDL Ports section of the Design Area to add a WSDL port.
  3. Connect the WSDL port to a JBI module or another WSDL port by doing the following:
    • Drag a Consume Endpoint on one component to a Provide Endpoint on another component.
    Refer to Connecting Composite Application Components for more information.
  4. Select the Edit icon on the WSDL port to view or edit its properties.
    You can also use the Navigator window for advanced editing of the composite application.
