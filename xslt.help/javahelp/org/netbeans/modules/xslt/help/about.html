The XSLT Designer provides a graphical environment for creating, deployment and testing XSL Transformation Services.

An XSL Transformation Service acts as a web service. It receives messages from a web service, transforms the messages in accordance with an XSL stylesheet and sends the messages back to the originator or forwards them to another web service.

With the XSLT Designer you can:

  • Create XSLT Module Projects
  • Create and edit XSL stylesheets with the XSL Transformation Editor
  • Deploy XSLT Module Projects (as part of a Composite Application Project)
  • Test XSL Transformation Services
