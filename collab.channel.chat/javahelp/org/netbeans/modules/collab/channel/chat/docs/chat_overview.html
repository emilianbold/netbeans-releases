About Chat

Once you are participating in a conversation, you have the ability to use the Chat channel to easily communicate with colleagues. The Chat channel is located at the bottom of the conversation window and has two main elements:

  1. The chat transcript pane shows the messages that have been sent within the conversation. All chat messages, including yours, will be displayed in the chat transcript.
  2. The chat input pane allows you to enter a message and then send it to the other participants in the conversation.

You can enter messages in the the chat input pane in several formats:

  • Plain text. Messages are sent as formatted text. This format is useful for sending general messages as well as stack traces or other formatted text content.
  • Java. Messages are sent as formatted and syntax-colored Java source code. This format is useful for quick discussions or reviews of Java source.
  • HTML. Messages are sent as HTML source. This format allows you to enter HTML and display it to the other participants as formatted and syntax-colored HTML.
  • XML. Messages are sent as XML source. This format allows you to enter XML and display it to the other participants as formatted and syntax-colored XML.
  If you copy Java source code, HTML source, or XML source from elsewhere within the IDE (or potentially other applications) and then paste it into the chat input pane, the message format automatically changes to the associated format. You can change the message format manually if you prefer to send the message in a different format than the one automatically selected for you.
