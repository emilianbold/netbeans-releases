JDBC Database Wizard: Select DataSource and Table Objects Page

See Also

The JDBC Database wizard allows you to quickly create a WSDL file for access to a table in a data source. The wizard also generates a corresponding XML Schema file that defines the elements for the table and columns referenced in the WSDL file.

The Select DataSource and Table Objects page contains the following fields:

  • Data Source. A drop-down list of all currently available data sources. Select a data source from the list to display the available tables from the data source.
    • Even if only one data source is listed, you must select it to display the available tables.
  • Available Tables and Selected Tables. Select a table from the list of displayed tables and move it to the Selected Tables list.
    • You cannot select multiple tables. The JDBC Database wizard only supports the selection of a single table.
