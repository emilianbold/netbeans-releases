Management Property File Wizard

This wizard allows you to generate a management configuration file. This file is a standard Java property file.
When launching a JDK 5.0 JVM, you can provide a management property file using the following System property
-Dcom.sun.management.config.file=<path to managemenet properties file>

To Access the wizard :
  1. Select File/New File...
  2. Select the Management category. In File Types select Management Properties File
  3. Type in a Name and a Location. Two more files will be generated. An access and password file that contain RMI authentication configuration.
  4. Click Next>
  5. Type in RMI related information. You can set an RMI Port, enable RMI Authentication, add Roles and Passwords, Enable SSL. 
  6. Click Next>
  7. You can enable SNMP and type in an UDP port, an SNMP interface, a trap port, enable Access Control, and set a custom ACL file.
  8. Click Next>
  9. You can then enable thread contention monitoring. 
  10. Finish> A popup window informs you to restrict access to password file. You have to change the generated .password file access. It must be readable for you only (On Unix systems : -rw-------). If you don't change the access, and you enable RMI Authentication, the Agent will not start.

    11.
Three files are generated in the selected location. You can use this generated properties file when running the Main Project with Remote Management (from menu Run/Management/Run(orDebug) Main Project with Remote Management...).
