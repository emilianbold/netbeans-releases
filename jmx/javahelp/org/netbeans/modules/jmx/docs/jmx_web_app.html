Accessing MBeans  from JSP

In this chapter we cover the steps needed to make a JSP access JMX management informations.

Using the JMX Agent and MBean Wizard

The use of these wizards is similar to its use in Java Application Project.
In the context of a web application, the JMX Agent is not runnable class. When generating the Agent, unselect the Create Main Method.

Implement your MBeans

Do it the usual way.

Create, name and register your MBeans within the Agent

Do it the usual way (in JMX Agent init method).

Display MBean informations in a JSP

Simply add the following line of code to make your JSP access  JMX MBean data :
<%--
    ...

    // Access to the JMX Agent. Doing so, your MBeans will be automaticaly registered.
    javax.management.MBeanServer server = <MyJMXAgent>.getDefault().getMBeanServer();

    // Get an MBean Attribute.
    Object obj = server.getAttribute(new ObjectName("AN MBEAN NAME HERE", "YOUR ATTRIBUTE NAME TO DISPLAY HERE");

    // Then implement your MBean Attribute value presentation...

    ...
--%>
