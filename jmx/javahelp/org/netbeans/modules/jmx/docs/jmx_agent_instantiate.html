Registering MBeans in a JMX Agent


The tasks involved to register MBeans in an MBeanServer can be achieved by calling menu action management | Instantiate and Register MBeans...

This action is contextual to a JMX Agent (generated from the JMX Agent wizard). A JMX Agent class node must be selected in the project tree.

  • MBean registration
  • Java object registered as MBean

The use of this action is not manadatory. You can manually update the agent source code in order to register MBeans.
