Standard Java SE Project Properties Dialog Box: Deployment

Use to enable a native packaging option in popup menu in the Projects window for J2SE projects.

You open the Standard Java SE Project Properties dialog box by right-clicking any standard Java SE project and choosing Properties.

Features Description

Enable Native Packaging Actions in Project Menu

When selected a Package As command is added to the context menu when you right-click the project node in the Projects window.
