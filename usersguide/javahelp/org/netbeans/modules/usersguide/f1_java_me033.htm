MIDP Project Properties: Obfuscating

Use to set the level the source code in a MIDlet suite is obfuscated for a specific project configuration.

Element Description

Use Values from "DefaultConfiguration"

Enables you to customzie the properties for a project configuration other than the default configuration if this option is unchecked.

Obfuscation Level

Sets the level of obfuscation.

Level Descripton

Describes the obfuscation level.

Additonal Obfuscation Settings

Adds additional obfuscation parameters.
