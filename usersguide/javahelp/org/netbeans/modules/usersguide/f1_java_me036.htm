MIDP Project Properties: Running

Use to configure options for running your project and setting security options.

Element Description

Use Values from "DefaultConfiguration"

Enables you to customize the properties for a project configuration other than the default configuration if this option is unchecked.

Emulator Command Line Options

Specifies command line options to run when the emulator is launched.

Regular Execution

Runs the project on an emulator device.

You can also set a security domain for the application to test its security.

Specify the Security Domain

Sets a security domain for the application to test its security.

This option only available in Regular Execution mode.

Execute Through OTA

Runs the project using Over-The-Air provisioning, which simulates the process of deploying an application from a server to a mobile device.

Debugger timeout

Sets the length of time (in milliseconds) that a launch waits to complete the launch process before terminating.
