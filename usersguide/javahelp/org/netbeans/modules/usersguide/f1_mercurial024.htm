Mercurial: Backout Dialog Box

Use the Backout dialog box to back out the effect of a commit.

You access this dialog box by selecting a versioned project, file or folder and either:

  • choosing Team > Recover > Backout from the main menu
  • right-clicking the selected item and choose Mercurial > Recover > Backout from the context menu

In the Backout dialog box, you have to set the following properties:

Element Description

Choose from Revisions

.Choose a changeset from the list.

Filter

.Specify filtering conditions to apply to the list of available changesets.

Details

.Displays the details of the selected revision.

Author

Name of a person who committed the revision.

Date

Date when the commit of the revision was made.

Description

Message specified during the commit of the revision.

Commit Msg

Specify a message used as a commit message for the backout commit.
