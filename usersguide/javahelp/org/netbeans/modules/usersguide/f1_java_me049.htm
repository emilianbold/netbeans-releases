New CDC Application Wizard: Select Platform Page

Use to choose the default emulator platform for the project.

Note: After you create the project, you can create additional project configurations with alternate platform settings.

Element Description

Java Platform

Specifies the execution environment for the project.

The platform you select determines which devices and profiles you can test your application on. The drop-down menu provides a list of available Java platforms. If the platform you want to develop for is not listed, you might need to download, install, and register the platform.

Device

Specifies an emulator supported by the selected Java platform.

When executed, the application's performance is simulated on this device.

Profile

Specifies the profiles supported by the selected Java Platform.

The drop-down menu provides a list of available profiles.
