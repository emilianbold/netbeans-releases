New Team Project Wizard: Name and Licenses

See Also

The New Team Project wizard enables you to create a project on a registered Team Server. You open the wizard by choosing Team Server > New Team Project from the Team menu. If you have an existing project that you want to share on a Team Server, see Sharing a Local Project.

This wizard enables you to create a project on a Team Server from within the IDE. When you create the project, the IDE sends the project information to the Team Server to create the project and Team Server services are assigned to the project.

  • You must be logged in to a Team Server to create a Team project.

In the Name and Licenses page you need to set the following properties.

  • Logged in to. If you are logged in, displays the user name. Click Login if you are not logged in to open the log in dialog box.
  • Team Project Name. The name that is used for the project URL on the Team Server website.
  • Project Title. The display name of the location. This name is used to identify the project in the IDE and the project page on the Team Server website.
  • Project Description. (optional) A description of the project. You can add or modify the description on the project webpage.
  • Project License. Select a license from the drop-down menu. You can modify the license on the project webpage.
