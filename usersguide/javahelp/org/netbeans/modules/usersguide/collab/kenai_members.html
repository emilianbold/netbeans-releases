Messaging Project Members

See Also

A Kenai project can have multiple members collaborating on a project. If the chat feature for the Kenai project is enabled and you are logged in to Kenai, the Kenai dashboard displays how many of the project members are online.

Chat Room

Each Kenai project can have an associated chat room open to members of the project. You can use the chat window for the project to exchange messages with other project members. The number of online members of projects and the number of unread messages is displayed in the status bar of the IDE. The names of the online members is displayed in a tooltip when you mouse over the chat icon in the status bar.

To open the project's chat window:

  • Choose Window > Chat Window from the main menu.
  • Expand the dropdown list and select the project.
