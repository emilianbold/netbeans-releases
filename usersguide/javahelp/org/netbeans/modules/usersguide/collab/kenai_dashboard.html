Using the Team Dashboard

See Also

The Team Dashboard enables you to quickly view information about Team projects that you have joined or opened. If you are logged in to a Team Server, the projects that you have joined are automatically listed in the Team Dashboard.

To open the Team Dashboard:

  • Choose Window > Team Dashboard from the main menu.

My Projects

You can join projects that you are interested in by bookmarking the project on the Team Server site. If a project is listed under your Open Projects in the Team window, you can click the Bookmark icon (  ) for the project to bookmark the project and add the project to the list of My Projects. When you log in to a Team Server in the IDE, all the projects that you have joined are listed under My Projects in the Team Dashboard. My Projects provides an overview that enables you to monitor and access each project's issues (  ) and messages (  ). To view more details about a specific project, click the Open Project icon (  ) to add the project to the list under Open Projects.

Open Projects

You can view additional details about a Team project by opening the project in the Team Dashboard. When you open a project, a node for the project is displayed under Open Projects. If you joined the project by bookmarking the project, the name is displayed in bold. Opening a project in the Team Dashboard does not automatically bookmark the project.

You can open a project in the following ways.

  • Clicking the Open Project icon of a project under My Projects in the Team Dashboard
  • Opening a project using the Open Team Project dialog box. To open the dialog box, choose Team > Team Server > Open Team Project from the main menu. Alternatively, click the Open Team Project icon (  ) at the top of the Team Dashboard.

To remove a project from the list of Open Projects, click the project's Close icon (  ).

You can expand a project listed under Open Projects to view the following details.

  • Chat Messages. The total number of new messages in the project's chat room is displayed next to the the Mesage icon (  ). Click the icon to open the chat window for the project.
  • Details. Click details to open a project tab in the main window that contains information about the project and links to project pages on Kenai.com, including links to the project wiki, download pages, mailing list archives, forums and repository changes.
  • Members. The Members node enables you to view a list of project members and see which members are online. You can double-click the name of a member to initiate a private chat.
  • Issues. The Issues node enables you to quickly view and submit project issues. You can work with issues directly in the IDE. For example, you can click New issue to open the New Issue tab in the main window and submit an issue.
  • Sources. The Sources node enables you to see the NetBeans projects associated with that Kenai project. Click Get to retrieve the source code to a local repository.
