Kenai Project Tasks: Quick Reference

The following table describes common Kenai project tasks.

To perform this task Follow these steps
Log in to Kenai.com
  1. From the main menu, choose Team > Kenai > Login to Kenai.

    You need to be registered at Kenai.com before you can login.
Add a local project to Kenai
  1. Right-click the project in the Projects window and choose Share on Kenai.
  2. Specify the details for the new Kenai project in the wizard. Click Finish.
Create a new Kenai project
  1. From the main menu, choose Team > Kenai > New Kenai Project.
  2. In the New Kenai Project wizard, specify the project name, title and license. Click Next.
  3. Specify the source code repository and issue tracking details.
  4. Review the details of the new project. Click Finish.
Open a Kenai project
  1. From the main menu, choose Team > Kenai > Open Kenai Project.
  2. Type the search terms. Click Search.
  3. Select the project in the list. Click OK.

    When you open a Kenai project, the project is listed under Open Projects in the Kenai dashboard. The project is not automatically bookmarked
View projects that you joined
  • Choose Window > Kenai from the main menu to open the Kenai dashboard.

    Projects that you joined are listed under My Projects.
View open projects in the Kenai dashboard
  • Choose Window > Kenai from the main menu to open the Kenai dashboard.

    Kenai projects that are open are listed under Open Projects. When a Kenai project is open in the dashboard, you can see an overview of the details of the project.

Retrieve sources of a project on Kenai
  1. Do one of the following:
    • Expand the Sources node of the project in the Kenai Dashboard and click "get".
    • Choose Team > Kenai > Get Sources from Kenai from the main menu.
  2. Select the Kenai Repository for the project.
  3. Specify the location for the local repository. Click Get From Kenai.
Log in to chat
  1. Choose Team > Kenai > Login to Kenai from the main menu.
  2. Enter your username and password in the Login to Kenai dialog box.
  3. Select Online on Chat. Click Login.

    If you are already logged in to Kenai, click the Offline icon (  ) in the stauts bar.
