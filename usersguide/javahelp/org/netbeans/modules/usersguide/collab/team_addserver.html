Adding a Team Server

See Also

To work with projects hosted on a Team Server in the IDE, you need to register the Team Server with the IDE. After you add the Team Server, you can log in to the Team Server to create, open and search for projects on the Team Server from the IDE.

To register a Team Server, perform the following steps.

  1. Open the Services window.
  2. Right-click the Team Servers node and choose Add Team Server.
  3. Specify the Team Server name and URL. Click OK.

When you click OK, a node for the Team Server is added below the Team Servers node in the Services window. After you register the Team Server, you can log in and use the Team Dashboard to view the details about your Team projects. If you register more than one Team Server, the Team Dashboard displays a drop-down list that enables you to select the current Team Server.
