GUI Builder: Property Editors

This dialog box allows to edit a value of the selected property in a more advanced or comfortable way than in the in-place editor in the Properties window. There are multiple ways (property editors) how to edit a value, which can be chosen from the combo box at the top. By default the basic property editor is chosen (for example, a text editor for String-type properties, or a color chooser for colors). Sometimes this default editor may have no graphics component (for example, numbers only have an in-place editor in the Properties window).

You can open the property editors dialog box for a currently selected element by clicking the ellipsis button (...) to the right of a selected component property in the Property window.

The following two property editors are always available for all value types:

Element Description

Value from existing component

Select a Component, Property, or Method Call to set the property value from.

Custom code

Insert your code representing the property value. The code becomes a part of the guarded code generated by the GUI Builder.
