Analyze Performance: Edit Profiling Roots (Advanced)

Use to access more advanced options for adding, editing and removing the root methods that are specified in the Edit Profiling Roots dialog box.

Element Description

Add JAR/Folde

Selects profiling roots that are located in a JAR or Java sources folder.

This option is only available when you are attaching the profiler to an external application

Add

Specifies a class and method that you want to instrument.

Edit

Modifies the selected class and root method.

Remove

Removes the selected root method from the list of methods that are instrumented.
