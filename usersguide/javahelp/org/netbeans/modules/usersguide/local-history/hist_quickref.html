Local History Tasks: Quick Reference

See Also

The following table describes common tasks that can be carried out using local history tools.

To perform this task: Follow these steps:
Create a file version
  1. Save (Ctrl-S) the file you are working on.
Compare two versions of a file side-by-side
  1. Right-click a file from the Projects or Files window and choose Local History > Show Local History. The Diff Viewer opens in the Source Editor.
  2. Select a version from the table in the upper pane of the Diff Viewer. The selected version displays on the left side in the Diff Viewer below. The right side of the Diff Viewer displays the current file version.
Revert a current file to a previous version
  1. Right-click a file from the Projects or Files window, and choose Local History > Revert to. The Revert To dialog box opens.
  2. Click on a row in the provided table that corresponds to the version you want to revert to. Click Revert.
Recover a previously deleted file
  1. From the Projects or Files window, right-click the project or directory that previously contained the file(s) which you deleted and choose Local History > Revert Deleted.
