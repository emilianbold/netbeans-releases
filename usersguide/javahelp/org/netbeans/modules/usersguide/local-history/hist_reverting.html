Reverting a File to a Previous Version

See Also

The IDE's local history functionality enables you to revert your current file to a previous version.

There are two ways to revert a file's current version to any previously stored version:

  • If you already know the version you want to revert to, select your file (e.g. from the Projects, Files, or Favorites window) and choose Local History > Revert to. The Revert To dialog opens, enabling you to select a version from the provided table.
  • If you want to view a version prior to selecting it, open the IDE's graphical Diff Viewer:
    1. Choose Local History > Show Local History from a file's right-click menu. The graphical Diff Viewer opens.
    2. Select a version from the table displayed in the upper area of the viewer. The selected version displays on the left side in the Diff Viewer below and is compared against your current version.
    3. After examining differences between the two versions, you can use the icons displayed in the Diff Viewer to make changes:
      • : Inserts the highlighted text from the previous revision into the current revision.
      • : Removes the highlighted text from the current version so that it mirrors the previous version.
      • : Reverts the file's current version to the state of the selected previous version.
  • You can make changes to your current version directly in the graphical Diff Viewer. To do so, place the cursor directly in the Current Version pane and edit where appropriate. The current version is instantaneously updated to reflect changes.
