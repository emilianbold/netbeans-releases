Local History: Diff Viewer

See Also

The IDE's Diff Viewer enables you to compare different versions of a file side by side using color coding to emphasize the differences between the files being compared.

The Previous () and Next () Difference buttons in the toolbar allow you to navigate among differences found between versions in the file.

The Diff Viewer highlights changes in the files using the following colors:

  • Blue. Indicates lines that have been changed since the earlier revision.
  • Green. Indicates lines that have been added since the earlier revision.
  • Red. Indicates lines that have been removed since the earlier revision.

The following icons allow you to make changes directly within the Diff Viewer:

  • : Inserts the highlighted text from the previous revision into the current revision.
  • : Removes the highlighted text from the current version so that it mirrors the previous version.
  • : Reverts the file's current version to the state of the selected previous version.
