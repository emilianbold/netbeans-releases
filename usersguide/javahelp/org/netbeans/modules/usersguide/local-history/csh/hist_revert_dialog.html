Local History: Revert To Dialog Box

See Also

The Local History Revert To dialog box enables you to revert a current version of a file to a previous version, as maintained in the IDE's local history repository.

You access this dialog box by right-clicking a file or folder in the Projects, Files, or Favorites window and choosing Local History > Revert to.

The dialog box is displays a two-column table presenting all previous versions of the selected file, listed in chronological order.

  • Version. Versions are arranged by date and named accordingly.
  • Label. Labels can be added to specific versions. To add a label to a version, click in the label text area for the corresponding version and type in a label.

To revert to a previous file version:

  • Select a version by clickin on the table row that it corresponds to. Click Revert.
