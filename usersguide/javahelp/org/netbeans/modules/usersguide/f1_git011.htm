Git Push to Remote Repository Wizard: Select Local Branches

The Select Local Branches panel of the Git Push to Remote Repository wizard enables you to specify what local branches need to be published to the remote repository.

To choose a branch to push to the remote repository, select it in the Local Branches list and click Next.
