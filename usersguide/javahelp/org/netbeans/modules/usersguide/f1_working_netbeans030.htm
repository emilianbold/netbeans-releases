Issue Tracking: Resolve Issue Dialog Box

You use this dialog box when you want to resolve an issue and change the status of the issue. You open this dialog box by right-clicking an issue entry in the Action Items window and choosing Resolve in the popup menu.

This dialog enables you to select a new status for the issue from the drop-down list and enter a comment to provide additional information about the change in status.
