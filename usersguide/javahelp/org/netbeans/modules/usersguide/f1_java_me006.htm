Add Libraries

Use to add libraries or resources to your project. You can also use this dialog to add or remove selected JAR and Zip files, library sources, and file folders with additional resources.

Element Description

Installed Libraries

Lists all libraries installed in the IDE.

Add Library

Adds the selected library to the project.

Edit

Opens the Library Manager where you can edit the library name or the library's classpath, sources, or Javadoc.
