Opening a Web Browser from the IDE

See Also

By default, the IDE opens your system's default web browser, however you can also specify a different browser if necessary.

To launch the web browser and open the specified home page.

  • Choose View > Web Browser.

Troubleshooting

If the IDE cannot find the operating system's default web browser, try one of the following:

  • Choose a different web browser.
    1. Choose Tools > Setup Wizard from the main menu.
    2. Select a different browser from the Web Browsers combo box.
  • Specify the location of the browser executable.
    1. Choose Tools > Options from the main menu.
    2. Expand IDE Configuration > Server and External Tools Settings > Web Browsers and select the browser type you are using.
    3. Open the property editor for Browser Executable property and set the full path to the executable in the Process field. You can click the ellipsis button (...) to browse to the executable.
