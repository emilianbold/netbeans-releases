Configuring a Web Browser

See Also

  1. Choose Tools > Options from the main menu
  2. Click General in the Options window.
  3. In the Web Browser drop-down, select the browser type you are using and click Edit.
  4. Enter the full path to the browser executable in the Process field. You can also use the ellipsis button (...) to browse to the executable.
  5. Enter any arguments in the Arguments field.
  6. Click Close.
