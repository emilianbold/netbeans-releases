Select Profiling Task: Filter Value Dialog Box

Use to view the classes that are to be profiled when you run the profiling session with the selected filter.

Element Description

To QuickFilter

 Adds all listed classes to the QuickFilter and the Set Quick Filter dialog box open.

Create a custom QuickFilter of specific classes by clicking To QuickFilter. The listed classes are added to the QuickFilter. In the Set Quick Filter dialog box that opens, you can choose the filter type and modify the list of filtered classes by entering the class or package names in the Filter Value text box. Choose Quick filter in the Filter drop-down list in the Basic settings pane of the Select Profiling Task window to apply the filter to the profiling session.
