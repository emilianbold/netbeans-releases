Profiler Window: Rename Snapshot Dialog Box

Use to change the name of a saved snapshot or heap dump. Select a snapshot in the Saved Snapshots section of the Profiler Control Panel and click Rename to open the Rename Snapshot dialog box. How?

Element Description

New File Name

Specifies the new name for the snapshot.
