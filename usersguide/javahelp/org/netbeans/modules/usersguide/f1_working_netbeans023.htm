Issue Tracking: Blocks Issue Dialog Box

You use this dialog box when you want to update an issue to indicate that the issue prevents the resolution of a specific existing issue. You open this dialog box by clicking the Search button (...) next to the Blocks text field in an issue's form.

Element Description

Select issue that this issue blocks

Enter the id or text contained in the summary of the issue that is blocked.

You can also select an issue from the dropdown list of issues that you recently viewed in the IDE.
