Documents Dialog Box

The Documents dialog box displays a list of the documents that are open in the editor and enables you to save, close or switch the editor focus to an open document. You open this dialog by choosing Window > Documents from the main menu.

The list of open documents is displayed in the left pane of the dialog box. When you select a document or documents in the list you can perform the following commands.

Element Description

Switch to Document

Click this button to make the document the active document in the editor. This button is disabled if multiple documents are selected.

Close Document(s)

Click this button to close the selected documents.

Save Document(s)

Click this button to save the selected documents. The names of documents that are not saved are displayed in bold.

When only one document is selected the location of the selected document is displayed in the Description textfield.

You can use the ctrl and shift keys to select multiple documents.

The names of documents that are under version control follow the color scheme that is specified for versioned documents in the Options window.
