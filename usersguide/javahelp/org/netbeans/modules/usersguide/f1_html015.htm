New HTML5 Application Wizard: Mobile Platforms Setup

You use this panel of the New HTML5 Application wizard to specify the details of the SDK for the mobile platform for the application. You see this panel when you create a PhoneGap application project and no PhoneGap SDK is specified in the Mobile Platforms tab in the Options window.

This panel contains the following elements:

Value Description

PhoneGap SDK Location

Specifies the location of the local installation of the PhoneGap SDK. You can use the Browse button to locate the folder that contains the SDK on your local system. You must specify a PhoneGap SDK to create a PhoneGap application.

If the SDK is not installed, you can click Download PhoneGap SDK to open the download page for the SDK in your browser.

Android SDK Location

Specifies the location of the local installation of the Android SDK. You can use the Browse button to locate the folder that contains the SDK on your local system.

If the SDK is not installed, you can click Download Android SDK to open the download page for the SDK in your browser.

iOS Code Sign Identity

Type the iOS code sign identity in the text field. This option is only enabled on Mac OS.

iOS Provisioning Profile

Select the iOS provisioning profile from the dropdown list. This options is only enabled on Mac OS.
