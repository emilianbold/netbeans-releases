CVS: Tag Dialog Box

The CVS Tag dialog box enables you to attach a tag to a file or folder. You can attach an existing tag or create a new tag.

To attach an existing tag to a file or folder, enter the tag name or click Browse to choose from a list of tags. To create and attach a new tag, enter the name of the new tag in the text field.

When working with tags, you can also specify the following options:

Element Description

Avoid Tagging Locally Modified Files

If selected, tags are not attached to local revisions that are not yet committed to the repository.

Move Existing Tags

If this option is selected and the specified tag is already attached, the tag is moved and attached to the current revision in your local working directory.

Delete Tag

If selected, the specified tag will be removed from the associated file.
