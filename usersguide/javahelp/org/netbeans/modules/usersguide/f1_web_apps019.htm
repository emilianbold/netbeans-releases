Standard Web Project Properties Dialog Box: Frameworks

You open the Standard Web Project Properties dialog box by right-clicking any standard web project and choosing Properties. In the Frameworks page, you choose one or more frameworks that you want to add to your web application.

Click the Add button and then choose one or both of the following:

Element Description

JavaServer Faces

Specifies that you want the IDE to add JavaServer Faces libraries to the project classpath.

Spring Web MVC

Specifies that you want the IDE to add Spring Framework libraries to the project classpath.

Struts 1.3.8

Specifies that you want the IDE to add Struts libraries to the project classpath.

Hibernate 3.2.5

Specifies that you want the IDE to add Hibernate 3.2.5 libraries to the project classpath and create a Hibernate configuration file (hibernate.cfg.xml) for your application. You can also specify a database connection (must be registered in the IDE), and the database dialect Hibernate uses to communicate with the database.
