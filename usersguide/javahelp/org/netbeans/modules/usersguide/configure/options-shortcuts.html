Options Window: Keymap

You use this panel in the Options window to set the keyboard shortcuts for the IDE. You can change between pre-configured shortcut sets, create a new shortcut set, or change the keyboard shortcut for an individual command.

Profile settings

You can create profiles to save different custom keymap settings. You can switch between profiles to quickly change between saved settings. If you modify either of the profiles included in the IDE, you restore the default settings by clicking Manage Profiles and then clicking Restore Manage Keymap Profiles dialog box.

You can import and export profile settings by clicking Manage Profiles and then clicking the Import or Export buttons in the Manage Keymap Profiles dialog box.

To create a profile:

  1. Click Keymap in the Options window.
  2. Click Manage Profiles next to the Profile drop-down list to open the Manage Keymap Profiles dialog box.
  3. Select the profile that you want to use as a base and click Duplicate.
  4. Type a name in the Create New Profile dialog box. Click OK.
  5. Select the new profile from the Profile dropdown list and modify the settings.
  6. Click OK to close the Options window.

You modify the shortcut for an action by performing the following steps.

  • Select an action in the list.
  • Double-click the shortcut to modify or add a shortcut.

As you press the key sequence, the correct syntax for that sequence automatically appears in the text field. If you simultaneously hold down the Alt key, the Control key, and the J key, "Alt+Ctrl+J" appears. To use a Backspace or Tab key in the shortcut, click the ellipsis button (...) and select the desired key. You can only specify a keyboard shortcut that is not being used by another command.

Notes:

  • To set keyboard shortcuts for recorded macros, click the Editor category in the Options window and then click the Macros tab.
