Customizing the Pop-Up Menu for an Object

See Also

In the IDE you can customize object pop-up menus to make the commands you frequently use more accessible.

To customize the pop-up menu for an object:

  1. From the main window, choose Tools > Options and click Advanced Options in the Options window.
  2. In the left pane of the Options window, expand IDE Configuration > System > Object Types.
  3. Select the desired object type.
  4. In the property sheet, click the ellipsis (...) button for the Actions property.

    The custom Property Editor is displayed. The left pane includes a hierarchical list of the commands in the IDE. The right pane shows the commands currently in the object type's pop-up menu.

  5. Change the pop-up menu as needed:
    • To add a command, expand the hierarchy in the left pane until you find the command you want, select the command, and click Add. The command is added to the end of the menu.
    • To remove a command, select the command in the right pane and click Remove.
    • To reorder the commands, choose a command in the right pane and click Move Up or Move Down.
    • To add a separator, click Add Separator. The separator is added to the end of the menu.
  6. Click OK.

    The IDE refreshes the object's pop-up menus, incorporating your changes.
