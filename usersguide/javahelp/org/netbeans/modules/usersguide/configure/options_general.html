Options Window: General

See Also

You use this panel to specify general proxy and auto-update settings and for specifying how source packages are displayed in the Projects window.

You can access this panel by choosing Tools > Options from the main menu, and then clicking General in the left pane of the Options window.

General settings

  • View Java Packages As. You can specify how Java source files are displayed in the Projects window by choosing one of the following view options:
    • List. When selected, files are displayed according to their logical structure.
    • Tree. When selected, files are displayed in a directory-based view.
  • Web Browser. The web browser that the IDE uses for displaying web content. Select one of the following:
    • <Default System Browser>. Your computer's default web browser.
    • Swing HTML Browser. An internal browser that provides a higher level of integration with the IDE than an external browser, but does not provide as much functionality.
    • Any other web browsers installed on your computer.

Proxy settings

  • Web Proxy. Specify the general proxy settings the IDE uses to access the Internet. If you select HTTP Proxy, you must specify Proxy Host and Proxy Port.
  • Proxy Host. Sets the host name or numeric IP address of the proxy server to use to connect to HTTP and FTP sites. To use this proxy you must select the HTTP Proxy checkbox.
  • Proxy Port. Sets the port number of the proxy server.

Auto Update settings

  • Check Period. Select how often the IDE checks for updates.
  • Ask Before Check. When selected, you are prompted before the IDE checks for updates.
