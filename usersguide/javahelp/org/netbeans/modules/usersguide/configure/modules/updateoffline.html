Installing Plugins Offline

See Also

If you have already downloaded a plugin's .nbm file, you can manually install it without having to connect to an update center.

To install a plugin offline:

  1. Choose Tools > Plugins from the main menu.
  2. Select the Downloaded tab and click Next.
  3. Click Add and browse to the location of the downloaded .nbm file. Select the file and click OK.
  4. Repeat the previous step for each plugin that you wish to add.
  5. Click Install complete the remaining pages of the wizard to install the plugin.
  • If the plugin that you have selected in the file chooser does not appear in the Downloaded tab of the Plugin Manager, you probably have the same or a newer version of that plugin installed already. You can check the Installed tab to verify that the plugin is installed and activated.
