Configuration Tasks: Quick Reference

See Also

The following table describes common Configuration tasks.

To perform this task Follow these steps
Configure general options
  1. Choose Tools > Options from the main menu.
  2. Select the category you wish to edit in the left pane.
  3. Specify the desired settings in the right pane.
Set the IDE's web browser
  1. Choose Tools > Options from the main menu.
  2. In the Options window, select General in the left pane and select the desired browser from the Web Browser drop-down list.
Configure proxy settings
  1. Choose Tools > Options from the main menu.
  2. In the Options window, select General in the left pane and select the Use HTTP Proxy Server check box.
  3. Enter the proxy host and proxy port.
Configure the Auto Update feature
  1. Choose Tools > Options from the main menu.
  2. In the Options window, select General in the left pane.
  3. Specify the update check frequency and if you want to be asked before the IDE checks for updates.
Install additional modules
  1. Choose Tools > Update Center from the main menu.
  2. Designate the location of modules you want to install in the Update Center wizard and click Next.
  3. Select the modules you wish to install.
  4. Review the licensing agreement and click Accept.
Customize the IDE's Menus
  1. In the Options window, click Advanced Options.
  2. In the tree in the left pane, expand IDE Configuration > Look and Feel.
  3. Right-click the Menu Bar node and choose Add > Menu.
  4. In the New Menu dialog box, type a name for the menu and click OK.

    The IDE adds an empty menu to the main window.

  5. Expand the Actions node and find the command you want to add to the menu, then right-click the command node and choose Copy.
  6. Expand the Menu Bar node, right-click the node of the menu you just created, and choose Paste > Copy.
Configure General Java Settings
  1. Choose Tools > Options from the main menu.
  2. Click Advanced Options in the Options window, expand the Editing node in the left pane and then select the Java Sources node.
  3. Set properties as desired in the right pane of the window.
Configure Editor Settings
  1. Choose Tools > Options from the main menu and select Editor in the left pane of the Options window.
  2. In the right pane, select the tab for the editor properties you want to edit.
  3. Set properties as desired and click OK.
