Plugins Manager: Available Plugins

You use the Plugins manager to dynamically update the IDE from the registered update centers and to manage IDE plugins. You use the Available Plugins tab of the Plugins manager to view and install new plugins available from the registered update centers.

In addition to the list of plugins, this panel contains the following elements:

Element Description

Search

Type a string in the text field to filter the list of displayed plugins.

Check for Newest

Click to check the update centers and refresh the list of available plugins.

Install

Click to install the selected plugin.
