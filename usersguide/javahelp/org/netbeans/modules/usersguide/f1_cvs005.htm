CVS: Edit CVS Root Dialog Box

The CVS Branch dialog box enables you to specify the CVS root for the repository when checking out files or importing files into the repository.

You can access this dialog box by clicking the Edit button in the CVS Root panel of the CVS Checkout wizard or the CVS Import Project wizard.

Element Description

Access Method

Select one of the following access methods from the drop-down list according to your type of CVS access:

pserver

For connecting to pserver repositories.

ext

For connecting to repositories using SSH, etc.

local

For protocol access to a local CVS repository. This type requires an external CVS executable.

fork

For direct access to a local CVS repository. This type requires an external CVS executable.

User

Enter the username for the repository.

Host

Enter the host name of the repository.

Port

Enter the port number if required.

Repository Path

Enter the path to the repository.
