Enterprise Application Client Project Properties Dialog Box: Packaging

Use the Packaging panel to set up the filter for files included in the JAR file. To open the Enterprise Application Client Project Properties dialog box, right-click any application client project node and choose Properties. Fromthe Build category, choose Packaging.

Notes:

To disable building of JAR files for a project, open build.xml, override the jar target and remove all the dependencies from the target, like so:

<target name="jar" />
