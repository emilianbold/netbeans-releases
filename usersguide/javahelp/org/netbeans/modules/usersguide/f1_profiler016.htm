Monitor Application: Advanced Options

Use to create a custom configuration and set more advanced profiling options.

Element Description

Override Global Settings

Enables you to override the global settings and specify the following options when profiling the application.

Working Directory

Specifies the directory to become current directory for your application when it starts. For example, you might have to specify a specific working directory when attaching to a local application.

Java Platform

Specifies a target JVM other than the default JVM the IDE is running on. You can choose any Java platforms registered in the Java Platform Manager.

JVM Arguments

Specifies any arguments to pass on to the JVM, for example, the default maximum heap size.
