Git: Select Revision Dialog Box

The Select Revision dialog box enables you to view the list of revisions maintained in the repository and choose the required revision.

The Select Revision dialog box contains the following elements:

Element Description

Branches

Expand this to select the required branch.

Commit ID

A unique identifier of the branched revision.

Author

Name of a person who committed the revision.

Message

A message specified during the commit of the revision.
