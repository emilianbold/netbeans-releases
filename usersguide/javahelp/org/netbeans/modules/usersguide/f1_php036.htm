Options Window: PHP: Frameworks & Tools: Composer

In this panel you can set global options for the Composer dependency management library.

Element Description

Composer

Specify the path to the Composer script. Use the Browse or Search buttons to locate the script on your local file system.

Vendor

Specify the name of the vendor directory. The default is vendor.

Author name

Specify the name that you want to appear as the author.

Author e-mail

Specify your email address in the text field.
