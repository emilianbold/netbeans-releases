Documents Dialog Box

The Documents dialog box displays a list of the documents that are open in the editor and enables you to save, close or switch the editor focus to an open document. The names of documents that are under version control follow the color scheme that is specified for versioned documents in the Options window.

The window contains the following elements.

Element Description

Documents

Select a document or documents from the list of open documents.

Switch to Document

Click this button to make the document the active document in the editor. This button is disabled if multiple documents are selected.

Close Document(s)

Click this button to close the selected documents.

Save Document(s)

Click this button to save the selected documents. The names of documents that are not saved are displayed in bold.

Order by

Select criteria for sorting the list of documents.

Description

Displays the location of the selected document. This field is empty when more than one document is selected.

You can use the ctrl and shift keys to select multiple documents.
