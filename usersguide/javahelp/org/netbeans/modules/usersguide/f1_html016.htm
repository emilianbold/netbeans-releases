Project Properties: Phonegap

In this panel you can modify the PhoneGap settings for your project.

You open this panel by right-clicking a project node in the Projects window and choosing Properties in the popup menu and selecting the PhoneGap category in the Project Properties window.

Value Description

Application ID

Type the id in the text field.

Application Name

Type in the text field to modify the name of the application.

Version

Select a tab to specify splash screen images for the platform.

Android

Specify the file that you want to use as the splash screen for each type of device.

iOS

Specify the file that you want to use as the splash screen for each type of device.

Setup Mobile Platforms

Click to open the Mobile Platforms tab in the Options window.
