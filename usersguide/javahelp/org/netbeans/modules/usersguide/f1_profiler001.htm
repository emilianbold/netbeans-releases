Attach Profiler: Select Process to Attach

Use to attach the profiler to a process that is running on your local system.

When you select a process, the dialog box displays details about the selected process such as the process ID (PID) and the main class.

Element Description

Process list

Enables you to select a process to attach the profiler to from a drop-down list.

Refresh

Updates the list of processes.
