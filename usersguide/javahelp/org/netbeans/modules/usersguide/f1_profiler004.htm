Select Profiling Task: Duplicate Configuration Dialog Box

Use to create a copy of an existing configuration for a profiling task. You can duplicate one of the default configurations or a custom configuration.

Element Description

Name

 Specifies the name for the duplicate copy of the selected configuration.
