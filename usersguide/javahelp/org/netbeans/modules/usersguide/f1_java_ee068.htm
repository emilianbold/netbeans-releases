glassfish-ejb-jar.xml Visual Editor: General

Use to specify the name of the EJB module.

The Visual Editor enables you to configure descriptor settings for an EJB module when your target server is the GlassFish server.

The Visual Editor groups descriptor settings under the following tabs:

Element Description

General

Enables you to edit the EJB module name.

EJB

enables you to set resource references in the EJBs in the module.

Security

Enables you to create and set properties for security roles in the EJB module

Web Services

Enables you to add web services used by the EJB and specify a WSDL publish location.

Messaging

Enables you to add JMS message destinations used by the EJB and specify the JNDI name of the destination.

XML

Enables you to view and edit the xml file directly in the xml editor.

Module Details

Element Description

Name

(Optional) Specifies the name of the module.
