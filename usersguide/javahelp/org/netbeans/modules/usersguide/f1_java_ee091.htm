Web Service Attributes Editor

For JAX-WS web services, the Web Service Attributes Editor helps you to customize various aspects of your application, in relation to its web services and clients. Depending on the context, the editor consists of one or more of the following panels:

Element Description

Quality of Service

The Quality of Service tab is for configuring components that enable interoperability between Java web services and .Net web services.

WSDL Customization

For web services and clients created from WSDL files, the editor provides the WSDL Customization panel. You can use the WSDL Customization panel to modify the artifacts generated from a WSDL file.

Wsimport Options

For JAX-WS client Web Service References, the Web Service Attributes Editor provides a tab for viewing and modifying the options for the wsimport and JAXB XML binding compiler (xjc) Ant tasks.

The table displays a list of the current import options. Click Add to add a new import option to the list.
