Mercurial: Select Changeset Dialog Box

Use the Select Changeset dialog box to view the list of revisions maintained in the repository and choose the required revision.

You access this dialog box by clicking Select Revision or Select Base in the Export Changesets dialog box (Team > Patches > Export Changeset Bundle).

In the Select Changeset dialog box, you have to set the following properties:

Element Description

Choose from Revisions

.Choose a changeset from the list.

Filter

.Specify filtering conditions to apply to the list of available changesets.

Details

.Displays the details of the selected revision.

Author

Name of a person who committed the revision.

Date

Date when the commit of the revision was made.

Description

Message specified during the commit of the revision.

Click Select to choose a changeset from the list of revisions.
