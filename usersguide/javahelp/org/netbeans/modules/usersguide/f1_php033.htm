Options Window: PHP: Hudson

In this dialog you set options for continuously testing PHP projects on a Hudson server. For more information about using Hudson with PHP, see .

Element Description

PPW Script

The path to the installed PPW (PHP Project Wizard) script. Use the Browse and Search buttons to locate PPW scripts on your local file system.

Job Config

The path to the config.xml file for Hudson or Jenkins PHP jobs. Use the Browse button to locate the configuration file on your local file system or click Default to use the config.xml that is included with the IDE (NETBEANS_INSTALLATION/php/hudson/config.xml).

The configuration file must include information about all relevant Hudson or Jenkins plugins. The IDE includes a config.xml with all Jenkins Template for PHP plugins. Alternatively, the administrator of your Jenkins or Hudson server can provide you with a file.
