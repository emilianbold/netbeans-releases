CVS: Branch Dialog Box

The CVS Branch dialog box enables you to specify the branch when creating a new branch or adding files to an existing branch.

Specify the following in the Branch dialog box:

Element Description

Branch name

Enter a branch name in the text field or click Browse to see a list of available branches. Entering a new branch name allows you to create a new branch.

When creating a new branch or moving files or folders to an existing branch, you can set the following options:

Element Description

Tag Before Branching

If selected, the name specified in the Tag Name text field is attached to the root of the branch. This can be useful for marking where the branch was made. The branch is created after the tag is attached to the file or folder.

Switch to This Branch Afterwards

If selected, the local working directory is switched to the specified branch after the branch is created.
