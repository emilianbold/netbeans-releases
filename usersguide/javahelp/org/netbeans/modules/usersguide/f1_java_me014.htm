Project Properties

Use to choose the active project configuration for your project. You can customize the properties for this configuration by choosing property sheets from the menu tree in the left pane of the Properties dialog.

Choose File > Project Properties (active_project_name) or right-click a project and choose Properties from the context menu to open this dialog box.

Element Description

Project Configuration

Specifies the current configuration.

Choose Add Configuration from the drop-down menu to add a new configuration to the project.

Manage Configurations

Displays the complete list of configurations created for the selected project.

You can add, duplicate, or remove configurations from the project, or create a project configuration template.

Category

Lists the applicable properties for the selected configuration.

The display to the right of the Category pane is dependent on the property selected.
