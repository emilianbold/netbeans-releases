Options Window: PHP: Frameworks & Tools: Symfony2

In this panel you can set the following options.

Element Description

Symfony

The path to the Symfony Standard Edition .zip archive. Use the Browse button to locate the archive on your local file system. If you do not have a copy of Symfony Standard Edition, click the download link. in the panel

Ignore cache directory by default

When selected the IDE does not include the /cache directory when scanning for Symfony functions. This option is selected by default. You might want to deselect his option if you cache actions.

This box globally sets whether cache directories are ignored. You can set whether the cache directory is ignored for an individual project in that project's Properties, under Frameworks > Symfony.
