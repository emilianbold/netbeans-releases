New Mobile Application Wizard: Install Platform Page

Use to create a CDC Java ME project.

The Install Platform page is called if there is no Java ME platform-compatible SDK or emulator platform, such as the Java ME SDK, installed in the IDE.

Element Description

Install SDK/Platform/Emulator

Opens the Platform Installation wizard, which enables you to add a platform.
