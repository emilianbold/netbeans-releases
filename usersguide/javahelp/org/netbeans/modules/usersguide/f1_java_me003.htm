Add/Edit Attribute

Use to add or modify project properties.

Element Description

Name

Lists the names of available attributes. Only Custom field names can be edited.

Value

Displays the values for each attribute.

Placement

Specifies the location of custom attributes. The default value is for attribute to be listed in both the JAD and JAR manifest files.

Some attributes can be changed in only in their properties pages. These properties are:

  • API permissions
  • Push Registry entries
  • MIDlet signing
