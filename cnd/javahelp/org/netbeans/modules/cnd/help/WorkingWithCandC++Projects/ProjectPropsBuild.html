C/C++ Project Properties Dialog Box: Build

The Build panel of the Project Properties dialog box lets you set the build tools properties for your project:

  • Development host
  • Tool collection
  • Platform
  • Whether the project requires a C compiler
  • Whether the project requires a C++ compiler
  • Whether the project requires a Fortran compiler
  • Whether the project requires an assembler
  • Configuration type
  • Whether make dependency checking is enabled
