Configure Code Assistance Wizard: View Configuration

See Also 

For a C/C++ Project With Existing Sources, the Configure Code Assistance wizard lets you specify how your code will be parsed by the built-in parser used for the code assistance features of the IDE.

The View Configuration page displays the options you have defined. These options are stored in the project properties for the active configuration of your project.

You can also view the include directories and preprocessor definitions that will be used for parsing the project in the Project Properties dialog box. Right-click the project and choose Properties, and expand the Parser Configuration node.
