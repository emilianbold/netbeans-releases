Configuring File Extensions for C/C++/Fortran Projects

See Also 

You can use the Options window to configure file extensions for your C and C++ projects.

To configure C and C++ file extensions:

  1. Choose Tools > Options.
  2. Click C/C++ in the top pane of the window.
  3. Click the Other tab.
  4. Modify the properties as desired.

You can set the following properties:

  • C File Extensions. The accepted extensions for C source files are listed with the default extension in bold type.
  • C++ File Extensions. The accepted extensions for C++ source files are listed with the default extension in bold type.
  • C/C++ Header File Extensions. The accepted extensions for C and C++ header files are listed with the default extension in bold type.
  • Fortran File Extensions. The accepted extensions for Fortran source files are listed with the default extension in bold type.
  • Assembler File Extensions. The accepted extensions for assembly source files are listed with the default extension in bold type.

For each list of extentions, you can click Edit to edit the list. In the Extensions List dialog box, you can add an extension, remove an extension, or change the default extension.
