New Remote Development Host Wizard: Setup Host

See Also 

Enter the user name and password that you use to log in to the remote host. If you do not want to specify the password each time you connect to the server through the IDE, select Remember Password. However, this option can compromise security.

Click Next after entering your password. The IDE then connects to the remote host, determines the host's platform, and searches for tool collections on your path on the remote host. When this process is complete, the Summary page is displayed.
