C/C++ Project Properties Dialog Box: Linker (General)

See Also 

The Linker (General) panel of the Project Properties dialog box let you specify: the following linker options for your C or C++ Application or Dynamic Library project.

  • linker output
  • additional library directories
  • runtime search directories
  • whether to strip symbols during linking
  • additional dependencies
  • the linking tool to be used

Use the buttons on the right side of the panel to open the dialog boxes where you can specify directories, dependencies, and tools. Use the checkbox to indicate whether you want symbols stripped.
