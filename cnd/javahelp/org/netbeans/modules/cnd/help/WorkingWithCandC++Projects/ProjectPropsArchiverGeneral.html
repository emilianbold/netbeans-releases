C/C++ Project Properties Dialog Box: Archiver

See Also 

The Archiver (General) panel of the Project Properties dialog box let you specify: the following archiver options for your C or C++ Static Library project.

  • archiver output
  • whether to run ranlib
  • whether to use verbose mode
  • whether to suppress diagnostics
  • additional dependencies
  • the archiving tool to be used

Use the buttons on the right side of the panel to open the dialog boxes where you can specify directories, dependencies, and tools. Use the checkboxes to indicate your preferences for ranlib, verbose mode, and diagnostics.
