C/C++ Project Properties Dialog Box: Compiler

See Also 

The C or C++ Compiler panel lets you set the following compiler options for your C or C++ Application project, Library project, or file:

  • include directories
  • preprocessor definitions
  • development mode
  • warning level
  • 64-bit architecture
  • whether to strip symbols during linking
  • tool to use for compiling

Use the buttons on the right side of the panel to open the dialog boxes where you can specify include directories, preprocessor definitions, and tools. Use the drop-down lists to set the development mode and warning level. Use the checkboxes to specify compilation for 64-bit architecture and stripping of symbols.
