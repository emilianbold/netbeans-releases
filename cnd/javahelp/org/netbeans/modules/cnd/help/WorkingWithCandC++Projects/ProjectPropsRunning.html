C/C++ Project Properties Dialog Box: Running

See Also 

The Running panel lets you specify run arguments and the run directory, set runtime environment variables, and specify whether to build the project before running it. Use the buttons on the right side of the panel to open dialog boxes where you can specify run arguments and the run directory, and set environment variables. Use the checkbox to specify whether to build the project.
