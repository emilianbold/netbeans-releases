Preprocessor Definitions Dialog Box

To add a preprocessor definition, do one of the following:

  • Click Add, type the preprocessor definition in the Add text field of the Edit dialog box, and click OK.
  • Select an existing preprocessor definition and click Duplicate. Then click Edit, modify the definition in the Edit dialog box, and click OK.

If you are setting preprocessor definitions for a file, definitions are inherited from the file's project by default. If you do not want the file to inherit preprocessor definitions from the project, click the Inherit from Project checkbox to remove the checkmark.
