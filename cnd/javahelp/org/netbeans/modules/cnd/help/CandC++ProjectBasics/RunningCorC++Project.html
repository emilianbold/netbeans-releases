Running a C or C++ Project

To run the main project:

  • Choose Run > Run Main Project.

To run an individual project:

  • Right-click the project's node in the Projects window and choose Run Project.

If the tools required to build and debug the project are not in the compiler collection specified for the project, the Resolve Missing Native Build Tools window opens to show which tools are missing. Correct the path to the missing tools and click OK.

If you are running a C/C++ project from existing sources and you did not specify a build result when you created the project, the Select Executable dialog box opens so that you can select the executable to run.
