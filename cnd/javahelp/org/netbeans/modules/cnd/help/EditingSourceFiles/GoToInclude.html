Going to an Included File

See Also

To go directly to a file that is included in a source or header file, do one of the following in the Source Editor:

  • Right-click in a #include directive in a source or header file and choose Navigate > Go to Declaration.
  • Ctrl-click in a #include directive.
The included file is opened in the Source Editor.
