Terminal Extensions


To Open Files in Editor from Terminal
  • Choose Window > IDE Tools > Terminal and navigate to the Project folder.

    • To open a single file, type the ideopenmodule.cc command.

    • To open multiple files, type the ideopen module.cc module.h command.

    • To open multiple files with wildcards that are supported by your current shell, type the ideopen module.* command.

Keyboard Shortcuts. Use Ctrl+Alt+Shift+T to open a new tab, and Alt+1, Alt+2 ... to switch between the tabs.

Accessing Hyperlinks from the Terminal. If a program's output prints a link to a file in the Terminal, and if you click on this link in Terminal, this file will be opened in the Editor. You can print a hyperlink in the terminal with an escape sequence.

fprintf(stdout, "(\033]10;%s;%s\007)]\n", "/home/../../CppApplication_48/main.cpp:20", "main.cpp:20")

Search in Terminal. To search in Terminal, use either Ctrl+Shift+F or Right-click and choose Find in Terminal option. You can also navigate between the found occurences. Use Shift+F3 to navigate to the previous occurrence and F3+Enter to navigate to the next occurrence. You can close the Find window by using the Esc key.
