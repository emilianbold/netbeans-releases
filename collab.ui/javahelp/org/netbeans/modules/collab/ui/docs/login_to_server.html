Logging Into the Collaboration Server

See Also

To start a collaboration session, you must first log into a collaboration server.

To log into the collaboration server:

  1. Choose Collaborate > Login.
    The collaboration login panel displays.
  2. In the collaboration login panel, select a collaboration account, provide the password, and click Login.
    A status message at the bottom of the panel will show the login progress.

Once you have logged in, the collaboration login panel will switch to the collaboration session view, which shows each session's list of contacts and conversations in tree form. To login to another account, repeat the login procedure.
