sun-web.xml Visual Editor: Add Principal Dialog Box

See Also

This dialog box enables you to specify the name of a principal to add to a security role. You can enter a new principal name or choose from the names of previously entered principals. You open this dialog box by clicking Add Principal in the Security Role Mapping pane of sun-web.xml or sun-ejb-jar.xml.

This dialog box enables you to add the following properties:

  • Principal Name. This field specifies the name of the principal assigned to a security role. The name of the principal must match a name specified on the Sun Java System Application Server/GlassFish.
  • Class Name. (Optional) This field specifies the implementation class.
