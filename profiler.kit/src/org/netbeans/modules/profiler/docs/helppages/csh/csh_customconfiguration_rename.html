Select Profiling Task: Rename Configuration Dialog Box

See Also

The Rename Configuration dialog box enables you to change the name of an existing custom configuration for a profiling task. You open this window by right-clicking the name of an existing custom configuration in the left pane of the Select Profiling Task dialog box and choosing Rename in the popup list.

To rename a custom configuration you need to specify the following options.

  • Name. Type the new name for the custom configuration.
