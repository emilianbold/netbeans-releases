Active Profiling Points Window

See Also

A profiling point is a marker in your source code which can invoke specific profiling actions. You set a profiling point in your code by using the popup menu in the Source Editor or by using the toolbar in the Profiling Points window.

The Active Profiling Points window displays a list of the profiling points that are currently active for the profiling session. You can open this window by clicking Show Active Profiling Points in the Select Profiling Task window.

  • You can also choose Window > Profiling > Profiling Points and view the active profiling points in the Profiling Points window.
