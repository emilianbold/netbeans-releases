CPU Snapshot: Select Settings Configuration Dialog Box

See Also

The Select Settings Configuration dialog box enables you to select an existing settings configuration for analyzing performance or create a new settings configuration. When you click OK in the dialog box the selected method will be added to the list of instrumentation root methods that will be used for that configuration setting.

You can have various configuration settings with different combinations of instrumentation root methods. When you analyze application performance you can select a configuration according to the root methods that you want to use for the profiling session.

You open this window by right-clicking a method name in a CPU snapshot and selecting Add to Instrumentation Roots in the popup menu. The text box lists the available configuration settings for the project. Select an existing setting or select <Create new configuration> to create a new custom configuration.
