Choose Heap Dump Destination Dialog Box

See Also

The Choose Heap Dump Destination dialog box enables you to specify where you want to save a .

You see this dialog box when you take a heap dump when you are running a profiling session. You can take a heap dump by clicking Dump Heap in the Profiler window. Alternatively, you can choose Profile > Take Heap Dump from the main menu.

In this dialog box you select one of the following locations for the heap dump.

  • Profiled project. When selected the IDE saves the heap dump to the default location in the directory for your project. This destination is selected by default.
  • Custom directory. Click Browse to specify a location on your local system for the heap dump.
