CVS: Commit Dialog Box

See Also

The CVS Commit dialog box enables you to commit local changes in version-controlled files to the repository.

You access this dialog box by right-clicking a version-controlled file or folder and choosing CVS > Commit.

The dialog box enables you to do the following:

  • Add a message to your commit. Use the Commit Message text area to add text describing the commit.
  • View files in commit. The files included in the commit are listed in the bottom pane of the dialog box. You can choose to exclude any of the listed files from the commit by selecting the file and then choosing Exclude from Commit from the drop-down list in the Commit Action column.
