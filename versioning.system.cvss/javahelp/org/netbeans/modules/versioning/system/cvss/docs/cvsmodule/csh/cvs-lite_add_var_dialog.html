Subversion: Add Variable Dialog Box

See Also

The Subversion Add Variable dialog box enables you to select CVS-reserved keywords used to indicate status labels for versioned files.

You access this dialog box by clicking the Add Variable button in the CVS Versioning Options window.

To select a status label, click on a listed label, then click OK. The selected label will be automatically entered into the Status Label Format text box of teh Versioning Options window.

The following CVS-reserved keywords are available as status labels:

  • Status. Specifies the current status of the file (Modified, New, etc.)
  • Tag. Specifies sticky tag information (branch name, tag name, date).
  • Binary. Specifies whether the file contains binary data.
  • Revision. Specifies the CVS revision number.
