CVS: Change CVS Root Dialog Box

See Also

The Change CVS Root dialog box enables you to modify the path to the CVS repository which you are working in. You can access this dialog box by selecting a versioned project or folder and choosing Team > (CVS >) Change CVS Root from the main menu.

For reference, the Change CVS Root dialog box displays the working copy on your system, as well as the CVS Root you are currently registered with.

Specify the following parameters in the dialog box:

  • New CVS Root. Enter a new root type and location. You can select a root location from the drop-down list (if previous entries have been made), otherwise type in a new root type and location. Alternatively, click the Edit button to open the Edit CVS Root dialog box.
  • Because the Versioning menu is context-sensitive, the Change CVS Root option is only active when you have previously selected a versioned project or folder.
  • Upon clicking Change, the new CVS Root is registered in the IDE, and will be applied the next time you attempt to connect to the repository.
