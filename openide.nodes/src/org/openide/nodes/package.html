The IDE uses nodes to represent JavaBeans or other property containers, formed into a hierarchical tree.

These nodes are used extensively for building the UI of the IDE, especially the Explorer. Using the Nodes API, it is possible to create a variety of custom nodes and entire subhierarchies, and manipulate and search existing nodes. In conjunction with the Datasystems API, nodes may also represent persistent data and can present the user interface to this data. Cookies may also be attached to nodes.
